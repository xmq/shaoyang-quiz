# 📚 Office 软件操作

使用说明：🔴红色 = 核心考点、判断改错点；🔵蓝色 = 得分关键词、公式与操作步骤；⚫️黑色 = 条件、版本差异、解释与例子。本章以 Windows 桌面版 Microsoft Office 为主；菜单名称会随版本变化，题干指定版本时以题干为准。

第一章：Word 文字处理软件

考点1：文件、视图与编辑状态

🔴【常见格式】：Word 2007 及以后默认格式为 `.docx`，Word 97—2003 常用 `.doc`；还可另存或导出为 `.pdf`、`.txt`、网页等。`.dotx` 是不含宏的模板，`.dotm` 是启用宏的模板。

🔴【常用视图】：

页面/打印布局视图：按打印页面显示页边距、页眉页脚等，适合排版；多数桌面版默认采用该视图。

阅读模式：侧重阅读，部分编辑工具会收起；不同版本名称和布局略有差异。

Web 版式视图：按网页方式连续显示，内容随窗口宽度重排。

大纲视图：按标题级别查看、折叠和调整长文档结构。

草稿视图：侧重快速编辑正文，不按完整打印页面呈现页眉页脚等对象。

⚫️【版本条件】：不能把“默认视图”“按钮位置”写成跨版本不变的结论；Word 2003、2010、2016、Microsoft 365 的界面和默认设置可能不同。

🔴【插入与改写】：插入模式在插入点处加入字符；改写模式会覆盖插入点后的字符。`Insert` 是否直接切换改写模式取决于 Word 选项和键盘，较新版本可能需要先启用该功能。

考点2：文本选择、删除与移动

🔴【鼠标操作必须说明位置】：

🔵正文文字区域：单击定位插入点；双击通常选中一个词；在同一段落内快速三击通常选中整个段落。

🔵页面左侧选定栏（正文左边距）：单击选中所指行；双击选中相邻段落；三击选中整篇文档。

🔵拖动可选择连续文本；按住 `Ctrl` 再选择可选取多个不连续区域；`Ctrl+A` 选中全文。

⚫️【判断陷阱】：题目只写“单击、双击、三击”，却不说明鼠标位于正文还是选定栏，结论是不完整的。触摸设备、Web 版 Word 的手势也不宜直接套用桌面版鼠标规则。

🔴【Backspace 与 Delete】：没有选中文本时，`Backspace` 通常删除插入点左侧字符，`Delete` 通常删除右侧字符；已经选中文本时，两者都可删除选中内容。

🔴【移动与复制】：拖动选中文本通常是移动；按住 `Ctrl` 拖动通常是复制。`Ctrl+X/C/V` 分别是剪切、复制、粘贴。

🔴【字号】：中文字号中一般“初号”大于“一号”，“二号”大于“四号”；以磅为单位时，磅值越大，字越大。不能混用两套序列直接比较而不看换算值。

考点3：段落、页面与节

🔴【段落格式】：对齐、缩进、段前/段后间距、行距属于段落格式；字体、字号、颜色、字符间距属于字符格式。

🔵【行距辨析】：单倍、1.5 倍、多倍行距会随字体等因素计算；“固定值”按指定磅值设置，字号过大时可能出现字符被遮挡；“最小值”至少保证指定值，必要时会自动增大。

🔴【分页符与分节符】：

分页符只强制从下一页开始，不创建新的节。

分节符把文档分成不同的节，可分别设置页边距、纸张方向、分栏、页码格式以及页眉页脚等。

🔵【不同页眉页脚步骤】：插入分节符 → 进入后一节页眉/页脚 → 关闭“链接到前一节” → 分别编辑。仅插入分页符不能实现独立设置。

⚫️【首页不同/奇偶页不同】：若只是同一节的首页或奇偶页需要不同页眉页脚，可启用相应选项，不一定要新增分节符。

🔴【分栏】：分栏属于页面布局，既可应用于选中文本，也可应用于当前节或整篇文档。可设置栏宽、间距和分隔线。

⚫️【数量陷阱】：可设栏数受 Word 版本、页面宽度、页边距和栏间距约束；“最多 11 栏”只可能是特定教材环境或默认页面参数下的口径，不应当作所有版本的绝对上限。

考点4：图文混排与对象

🔴【嵌入型与浮动对象】：嵌入型图片像一个字符一样随文本排列；其他环绕方式属于浮动对象，可相对页面、段落或字符定位。

🔴【常见环绕方式】：四周型按对象外接矩形环绕；紧密型更贴近对象轮廓；衬于文字下方常用于背景；浮于文字上方可能遮挡文字。

⚫️【默认方式有条件】：很多桌面版 Word 的“插入图片”默认采用嵌入型，但用户可在选项中修改默认环绕方式，粘贴来源也会影响结果。题干未指定环境时，应说“通常/默认设置下”。

🔴【等比例绘制】：拖动矩形或椭圆时按住 `Shift` 通常约束为正方形或正圆；按住 `Ctrl` 常用于从中心绘制。不同工具、触控方式可能略有差异。

🔴【组合条件】：通常只能组合浮动对象；嵌入型对象需先改为非嵌入型环绕。组合后可整体移动、缩放和设置格式。

考点5：表格

🔴【选择对象决定操作结果】：

选中单元格内容后按 `Delete`，通常只清除内容，不删除单元格结构。

使用“删除单元格/行/列/表格”命令，才会改变表格结构；选中整张表并执行“删除表格”可删除表格和内容。

🔵【拆分与合并】：合并单元格是把相邻单元格合为一个；拆分单元格是把一个单元格分成多个；拆分表格会在指定行前把一张表分成两张。两张相邻表格能否重新连接，取决于它们之间的段落标记、环绕方式和结构，不能笼统写成“拆分后一定可直接合并”。

🔴【表格计算】：Word 表格可使用 `SUM`、`AVERAGE` 等简单公式，例如 `=SUM(ABOVE)`；修改源数据后往往需要手动更新域（如选中后按 `F9`），不像 Excel 那样普遍实时重算。

🔴【重复标题行】：跨页表格可将首行设置为标题行重复出现；若表格被手动分页或嵌套，效果可能受限。

考点6：查找替换、样式与自动化

🔴【查找与替换】：可查找/替换文本、格式、通配符和多种特殊字符。桌面版还可用特殊代码查找某些嵌入式图形；浮动形状及复杂对象的查找能力有限。因此“只能替换文字”或“绝对不能查找图形”都不严谨。

🔴【格式刷】：单击格式刷通常使用一次；双击可连续应用；按 `Esc` 或再次单击退出。格式刷复制的是格式，不复制正文内容。

🔴【样式】：样式是一组命名格式，可统一标题与正文外观，也是自动目录、大纲导航和长文档结构化的基础。手工把字放大加粗不等于设置了“标题样式”。

🔴【目录】：先正确应用标题样式，再插入自动目录；标题或页码变化后需要更新目录，而不是手工逐项修改。

🔴【修订与批注】：修订记录插入、删除和格式更改，审阅者可接受或拒绝；批注用于讨论和备注，通常不直接改正文。删除批注不等于接受修订。

🔴【邮件合并】：由主文档、数据源和合并域构成，可批量生成格式相同而个别字段不同的信函、标签、证书等。

🔴【宏】：宏是为自动执行一系列操作而记录或编写的命令序列，常以 VBA 过程实现；“宏是一段应用程序”过于笼统。宏功能本身不是病毒，但不可信文档中的宏可能携带恶意代码。

第二章：Excel 电子表格软件

考点1：工作簿、工作表与单元格

🔴【常见格式】：Excel 2007 及以后默认 `.xlsx`；旧格式为 `.xls`；含 VBA 宏的工作簿常用 `.xlsm`；模板常用 `.xltx`。

🔴【工作簿与工作表】：工作簿是文件，工作表是工作簿内的表页。新建工作簿默认工作表数量可由版本和用户设置决定：Excel 2013 及以后通常默认 1 张，较早版本常见 3 张，不能统一写成“永远 3 张”。

⚫️【数量上限】：现代 Excel 可添加的工作表数量主要受可用内存限制；“最多 255 张”不是现代版本的固定上限。旧版设置中“新工作簿内工作表数”可选到 255，也不等于所有工作簿只能有 255 张。

🔴【单元格地址】：`B3` 表示 B 列第 3 行。活动单元格是当前输入位置，通常只有一个；一个选区可以同时包含许多单元格。

🔴【现代工作表规模】：Excel 2007 及以后每张工作表为 1,048,576 行、16,384 列，最后一列为 `XFD`；Excel 2003 及以前为 65,536 行、256 列。

考点2：数据输入、显示与填充

🔴【长编号】：身份证号、学号等不参与运算的编号应按文本存储，可先设为“文本”，或在输入时加英文单引号。Excel 对数值仅保证约 15 位有效数字，直接当数值输入可能导致末位变为 0。

🔴【`######`】：最常见原因是列宽不足；日期或时间结果为负数等情况下也可能显示一串 `#`。因此应先结合单元格内容和格式判断，不能只认定为列宽问题。

🔴【单元格内换行】：Windows 桌面版通常用 `Alt+Enter`；还可启用“自动换行”。

🔴【自动填充】：可复制常量和公式，也可扩展日期、星期、等差序列及自定义序列。公式被填充时，引用会按相对/绝对规则变化。“不能填充公式”是错误说法。

⚫️【自动识别有环境条件】：输入 `1/2` 可能按当前区域设置识别为日期，也可能按格式或区域设置产生不同结果；要保留字符串应设为文本。公式必须以 `=` 开头，`=2/5` 才执行除法。

考点3：引用、运算符与错误值

🔴【相对引用】：`A1`，复制公式时行列按相对位置改变。

🔴【绝对引用】：`$A$1`，复制公式时行列都锁定。

🔴【混合引用】：`$A1` 锁列不锁行；`A$1` 锁行不锁列。编辑公式时可用 `F4` 在引用形式间切换（部分笔记本需配合 `Fn`）。

🔴【跨表引用】：`Sheet2!$A$4` 表示 Sheet2 中 A4 的绝对引用；工作表名含空格时需写成 `'期末成绩'!A1`。

🔴【运算优先级】：引用运算 → 负号/百分号/幂 → 乘除 → 加减 → 连接 `&` → 比较；有疑问时用括号明确顺序。

🔴【常见错误值】：`#DIV/0!` 除数为 0 或空；`#VALUE!` 数据类型不合适；`#NAME?` 名称或函数名无法识别；`#REF!` 引用无效；`#N/A` 没有可用值；`#SPILL!` 是动态数组结果区域被占用等（新版 Excel）。

考点4：函数与公式

🔴【公式规则】：公式以 `=` 开头；函数由函数名、括号和参数组成。文本条件通常需要英文双引号。

🔵【高频函数】：

`SUM(B2:B5)` 求和；`AVERAGE(B2:B5)` 求算术平均；`MAX/MIN` 求极值；`COUNT` 统计数值单元格个数；`COUNTA` 统计非空单元格个数。

`COUNTIF(B2:B8,">=60")` 单条件计数；`SUMIF` 单条件求和；`COUNTIFS/SUMIFS` 处理多条件。

`IF(B2>=60,"及格","不及格")` 条件判断；新版可使用 `IFS`，但旧版本不支持。

`ROUND(A1,2)` 四舍五入保留两位小数；`LEFT/RIGHT/MID` 提取文本；`LEN` 统计字符数；`TODAY()` 返回当前日期。

⚫️【范围边界】：`A1:A4` 包含 A1、A2、A3、A4；冒号表示连续区域，逗号在多数环境中分隔参数（受区域设置影响，部分环境使用分号）。

考点5：排序、筛选、汇总与透视表

🔴【排序】：可按一个或多个关键字排序。排序连续数据区域时应让相关列一起扩展选择，否则可能破坏记录之间的对应关系。

🔴【筛选】：筛选通常隐藏不符合条件的整行，不删除记录；清除筛选条件可恢复显示。高级筛选可处理较复杂的条件区域并可将结果复制到别处。

🔴【分类汇总】：使用传统“分类汇总”命令前，应先按分类字段排序，使同类记录相邻；“全部删除”可移除分类汇总。是否能撤销取决于后续操作和应用状态，不能写成“删除后绝对不能撤销”。

🔴【数据透视表】：用于对大量明细按字段分组、汇总、交叉分析；源数据变化后通常需要刷新数据透视表，不能与普通图表“自动实时更新”混为一谈。

🔴【数据有效性/数据验证】：可限制允许输入的数据类型或范围、创建下拉列表并设置出错警告；它不能代替密码保护。

考点6：图表、打印与保护

🔴【图表选择】：柱形/条形图适合比较类别；折线图适合展示随时间变化的趋势；饼图适合少量类别构成比例，且通常只表达一个数据系列；散点图适合研究两个数值变量关系。

⚫️【图表联动】：普通图表引用的源数据变化后通常会更新；若新增数据不在原引用区域内，图表未必自动扩展。把数据区域转换为“表格”可提高自动扩展能力。

🔴【冻结窗格】：用于滚动时保持标题行/列可见；不是锁定编辑。保护工作表用于限制对锁定单元格等内容的修改，但不等同于强加密。

🔴【打印】：打印区域、分页、页边距、缩放和重复打印标题行是高频设置；“将工作表调整为一页”可能使内容过小，应区分“1 页宽”和“1 页高”。

第三章：PowerPoint 演示文稿软件

考点1：文件、页面与视图

🔴【常见格式】：`.pptx` 为普通演示文稿，`.ppt` 为旧格式，`.ppsx` 打开时通常直接进入放映，`.potx` 为模板；含宏文件分别常用 `.pptm/.ppsm/.potm`。

🔴【演示文稿与幻灯片】：演示文稿是文件，幻灯片是其中的一页。单页可复制到新演示文稿，也可导出为图片或 PDF 页面，因此“幻灯片绝对不能单独保存”不严谨。

🔴【常见视图】：普通视图用于编辑；幻灯片浏览视图适合整体排序；备注页视图编辑讲者备注；阅读视图在窗口中播放。不同版本是否单列“大纲视图”有所差异，较新版本常在普通视图的大纲窗格中使用。

考点2：编辑与放映

🔴【`Ctrl+A` 有对象范围】：焦点在幻灯片缩略图窗格时选中所有幻灯片；焦点在当前页编辑区时选中当前页的全部对象；焦点在文本框内时选中该文本框文字。不能只写“Ctrl+A = 全选幻灯片”。

🔴【放映快捷键】：Windows 桌面版通常 `F5` 从头放映，`Shift+F5` 从当前幻灯片放映，`Esc` 退出。笔记本可能需要配合 `Fn`，网页版/移动版快捷键可能不同。

🔴【复制幻灯片】：可用复制粘贴或“复制幻灯片”；在缩略图/浏览视图中按住 `Ctrl` 拖动通常可复制，单纯拖动通常是移动。

🔴【隐藏幻灯片】：放映时跳过该页，但页面仍保留在文件中；并非删除。

考点3：主题、母版与版式

🔴【主题】：统一定义颜色、字体、效果和背景样式；模板是在主题基础上还可包含版式、占位符和示例内容的文件。

🔴【幻灯片母版】：控制一组版式的共同格式和占位符。母版不是 `.potx` 文件；`.potx` 是演示文稿模板的扩展名。

⚫️【母版修改范围】：修改母版通常影响使用该母版/版式且未被单页覆盖的幻灯片；单页手工格式、被删除的占位符或多母版演示文稿可能不会完全同步。因此“母版一改，所有对象无条件变化”是错误说法。

🔴【多主题/多母版】：一个演示文稿可以包含多个幻灯片母版，从而让不同部分使用不同主题或版式。

考点4：动画、切换与交互

🔴【动画与切换】：动画作用于一张幻灯片内的对象，常见类别为进入、强调、退出、动作路径；切换作用于幻灯片之间。二者对象不同，不能混淆。

🔵【动画触发方式】：单击时、与上一动画同时、上一动画之后；还可设置持续时间、延迟、重复和触发器。动画窗格用于调整顺序和参数。

🔴【超链接】：超链接可附着在文字、图片、形状等对象上；目标可以是网页、文件、电子邮件地址、本演示文稿中的指定幻灯片，也可在支持的桌面版本中定位到另一演示文稿中的特定幻灯片。原稿“不能链接到某个对象/另一演示文稿某一页”表述错误。

🔴【动作按钮】：可设置跳转、运行程序/宏、播放声音等动作；“鼠标单击”和“鼠标移过”是不同触发方式。

考点5：媒体、打包与演示规范

🔴【音视频】：现代 PowerPoint 可插入音频和视频，并设置自动播放、单击播放、跨幻灯片播放、循环等；可用格式与编解码器、Office 版本和操作系统有关。

🔴【字体与链接资源】：把文件交给另一台电脑时，应检查特殊字体、外部链接的视频/音频和文件路径；较新版本通常会嵌入部分媒体，但外部链接资源仍可能丢失。可使用“打包演示文稿”等功能降低缺失风险。

🔴【演示者视图】：演讲者可查看当前页、下一页、备注和计时，观众只看到放映画面；需要正确配置多显示器。
