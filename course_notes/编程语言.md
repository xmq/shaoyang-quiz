# 编程语言基础（C + Python 3）

## 一、程序与编程语言
<!-- exam: B | 单选、判断 -->

### 1. 程序和算法

程序是用计算机能够执行或翻译的形式表示的一组指令。算法是解决问题的明确步骤。

算法常见基本性质：

- 有穷性：经过有限步骤后结束。
- 确定性：每一步含义明确。
- 可行性：每一步能够执行。
- 输入：可以有零个或多个输入。
- 输出：至少产生一个结果。

程序不等于算法。算法强调解决问题的步骤，程序是算法用具体语言实现后的形式。

### 2. 三种基本结构

- 顺序结构：语句按书写顺序依次执行。
- 选择结构：根据条件选择不同分支。
- 循环结构：在条件满足时重复执行某段语句。

结构化程序通常由这三种结构组合而成。

### 3. 编译与解释

编译通常先把整个源程序翻译成目标程序，再执行；解释通常在运行过程中逐句翻译并执行。现代语言实现可能同时使用编译、字节码和虚拟机，因此“编译型”和“解释型”不是所有实现的绝对二分。

C语言通常先编译再运行；Python常先生成字节码，再由解释器或虚拟机执行。

### 4. 程序错误

- 语法错误：违反语言书写规则，通常不能正常翻译或运行。
- 运行时错误：程序运行过程中发生，如除零、无效下标。
- 逻辑错误：程序能够运行，但结果不符合要求。

测试用于发现错误，调试用于定位并修改错误。

## 二、标识符、变量与数据类型
<!-- exam: B | 单选、判断、填空 -->

### 1. 标识符

标识符用于给变量、函数等命名。C和Python常见规则是由字母、数字、下划线组成，不能以数字开头，不能使用关键字，并且区分大小写。

例如 `score_1` 合法，`1score` 不合法；`name` 与 `Name` 是不同标识符。

### 2. 变量与常量

变量用于保存可变化的数据，通常有名称、值和一定的作用范围。常量是在规定范围内不改变的数据。

C变量通常在使用前声明类型，例如：

```c
int age = 20;
float score = 85.5f;
```

Python变量通过赋值建立名称与对象的绑定：

```python
age = 20
score = 85.5
```

### 3. 基本数据类型

- 整数：表示没有小数部分的数。
- 浮点数：近似表示带小数的数据。
- 字符：表示单个字符，C常用 `char`。
- 字符串：表示字符序列。
- 布尔：表示真和假，Python用 `True/False`；C条件中常用0表示假、非0表示真。

浮点数采用有限位二进制近似表示，部分十进制小数不能精确存储，因此比较时要注意误差。

### 4. 类型转换

隐式转换由语言自动完成，显式转换由程序员指定。把范围更大或精度更高的数据转成较小类型时，可能发生溢出或精度损失。

## 三、运算符与表达式
<!-- exam: A | 单选、判断、填空、程序阅读 -->

### 1. 算术运算

常见算术运算符有加、减、乘、除和求余。

- C中两个整数相除，结果仍按整数处理，小数部分被截去。
- Python中 `/` 是普通除法，`//` 是整除，`%` 是求余。

例如C语言 `7/2` 的结果为3；Python `7/2` 为3.5，`7//2` 为3。

### 2. 关系运算

关系运算用于比较大小或相等，结果为真或假。常见运算符有 `> < >= <= == !=`。

判断陷阱：C和Python中 `=` 是赋值，`==` 才是判断相等。

### 3. 逻辑运算

C使用 `&&`、`||`、`!` 表示与、或、非；Python使用 `and`、`or`、`not`。

逻辑与要求两侧都为真，逻辑或只要一侧为真，逻辑非把真假取反。与、或常采用短路求值：结果已经确定时，右侧表达式可能不再计算。

### 4. 优先级

一般先算括号，再按运算符优先级计算。不确定时应使用括号明确顺序，不要只凭记忆猜测复杂表达式。

## 四、输入、输出与顺序结构
<!-- exam: A | 单选、填空、程序阅读 -->

### 1. C语言输入输出

`printf` 用于格式化输出，`scanf` 用于格式化输入。

```c
int x;
scanf("%d", &x);
printf("%d\n", x + 1);
```

常见格式：`%d` 整数，`%f` 浮点输出，`%c` 字符，`%s` 字符串。使用 `scanf` 读取普通变量时通常要传变量地址。

### 2. Python输入输出

`input()` 读取一行文本，返回字符串；`print()` 输出内容。

```python
x = int(input())
print(x + 1)
```

如果需要数值计算，应使用 `int()` 或 `float()` 转换输入。

### 3. 简单输出跟踪

```python
x = 3
y = x + 2
x = y * 2
print(x, y)
```

执行顺序为：`x=3`，`y=5`，再把 `x` 改为10，因此输出 `10 5`。读程序时要逐句更新变量，不要把变量初值和最终值混在一起。

## 五、选择结构
<!-- exam: A | 单选、判断、程序阅读 -->

### 1. C语言选择

```c
if (score >= 60) {
    printf("pass");
} else {
    printf("fail");
}
```

C用圆括号写条件，用大括号组织语句块。多个条件可使用 `else if`。

### 2. Python选择

```python
if score >= 60:
    print("pass")
else:
    print("fail")
```

Python在条件后写冒号，并用缩进表示代码块。多个分支可使用 `elif`。

### 3. 分支判断方法

先按当前变量值计算条件真假，只执行被选中的分支。多分支题还要注意条件顺序，范围更大的条件写在前面可能使后面的分支永远无法执行。

## 六、循环结构
<!-- exam: A | 单选、判断、填空、程序阅读 -->

### 1. while循环

`while` 在每次执行循环体前先判断条件，因此可能一次也不执行。

```python
i = 1
s = 0
while i <= 3:
    s = s + i
    i = i + 1
print(s)
```

循环依次把1、2、3加入 `s`，输出6。

### 2. for循环

C语言常用初始化、条件和更新三部分：

```c
for (int i = 0; i < 3; i++) {
    printf("%d", i);
}
```

输出 `012`。

Python常遍历序列或 `range`：

```python
for i in range(3):
    print(i)
```

`range(3)` 依次产生0、1、2，不包含3。

### 3. break与continue

- `break` 立即结束当前最近一层循环。
- `continue` 跳过本轮剩余语句，进入下一轮。

它们不会自动结束整个程序，也不会同时结束所有嵌套循环。

### 4. 循环跟踪法

列出“轮次、循环变量、条件结果、关键变量、输出”五项，逐轮记录。检查初始化、循环条件、变量更新和停止时的值，是读循环题最稳的方法。

## 七、函数与作用范围
<!-- exam: A | 单选、判断、程序阅读、场景题 -->

### 1. 函数

函数是完成特定任务、可以被调用的程序单元。作用包括分解问题、减少重复代码、提高可读性和便于维护。

### 2. 参数与返回值

形参写在函数定义中，实参写在调用位置。返回值是函数把结果交给调用者的方式。

C语言示例：

```c
int add(int a, int b) {
    return a + b;
}
```

Python示例：

```python
def add(a, b):
    return a + b
```

### 3. 局部变量和全局变量

局部变量主要在所属函数或代码块内使用；全局变量作用范围较大。局部变量有助于减少相互影响，全局变量过多会增加程序理解和维护难度。

C语言参数按值传递，修改普通形参通常不改变调用者变量。Python传递对象引用，函数内重新绑定形参也不等于重新绑定调用者名称。

## 八、数组、字符串与Python容器
<!-- exam: A | 单选、判断、程序阅读、场景题 -->

### 1. C数组和字符串

数组保存一组相同类型的元素，常从下标0开始。长度为5的数组有效下标通常为0到4。

```c
int a[3] = {10, 20, 30};
printf("%d", a[1]);
```

输出20。C字符串是以空字符 `\0` 结束的字符数组，空间应给终止符留出位置。

### 2. Python列表和元组

列表有序、可修改，使用方括号；元组有序、通常不可修改，常使用圆括号。

```python
a = [10, 20, 30]
a[1] = 25
print(a[1])
```

输出25。

### 3. 字典和集合

字典保存“键—值”映射，适合按键查找；集合保存不重复元素，适合去重和集合运算。

```python
person = {"name": "Li", "age": 20}
print(person["age"])
```

输出20。

### 4. 切片

Python切片 `a[start:stop]` 取从start开始到stop之前的元素，不包含stop。

```python
a = [0, 1, 2, 3]
print(a[1:3])
```

输出 `[1, 2]`。

## 九、C指针、结构体与文本文件
<!-- exam: A | 单选、判断、填空、程序阅读、场景题 -->

### 1. 指针基础

指针是保存内存地址的变量。`&`用于取得变量地址，`*`用于通过指针访问其所指对象，称解引用。

```c
int x = 10;
int *p = &x;
*p = 20;
printf("%d", x);
```

`p`保存`x`的地址，`*p=20`修改该地址中的值，因此输出20。声明中的`int *p`表示p指向int对象；表达式中的`*p`表示访问所指值。

空指针不指向有效对象，不能直接解引用；未初始化指针的内容不确定，也不能拿来访问内存。基础题先分清“变量值、变量地址、指针保存的地址、解引用后的值”。

数组名在许多表达式中会转换为首元素地址，但数组和指针不是完全相同的对象。基础判断只需知道指针可用于访问数组元素，不做复杂指针运算。

### 2. 结构体

结构体把若干可以不同类型的数据组合成一个整体，适合表示学生、设备等具有多个字段的对象。

```c
struct Student {
    int id;
    float score;
};

struct Student s = {1, 85.0f};
printf("%d", s.id);
```

结构体变量用`.`访问成员，所以输出1；结构体指针通常用`->`访问成员。结构体与数组不同：数组元素类型相同，结构体成员可以类型不同。

### 3. C文本文件

C使用`FILE *`表示文件流，常用`fopen`打开、`fclose`关闭；`fprintf/fscanf`按格式写读，`fgets/fputs`按文本行或字符串读写。

```c
FILE *fp = fopen("score.txt", "w");
if (fp != NULL) {
    fprintf(fp, "%d", 90);
    fclose(fp);
}
```

`"r"`用于读取，文件应已存在；`"w"`用于写入，通常会创建文件或清空原内容；`"a"`在文件末尾追加。打开后要检查是否成功，使用完应关闭文件。

## 十、Python异常与文本文件
<!-- exam: B | 单选、判断、程序阅读、排障题 -->

### 1. 异常处理

异常是程序运行时出现的非正常情况，如数值转换失败、文件不存在或除数为零。`try`放可能出错的语句，`except`处理指定异常，`else`在没有异常时执行，`finally`无论是否异常通常都会执行。

```python
try:
    x = int("abc")
except ValueError:
    print("error")
```

字符串不能转换为整数，因此捕获`ValueError`并输出`error`。异常处理用于有计划地处理可预见问题，不应使用空的`except`隐藏所有错误。

### 2. with open

`open()`打开文件，常见模式同样有`"r"`读取、`"w"`覆盖写入、`"a"`追加。`with`语句可在代码块结束后自动关闭文件，即使发生异常也能完成资源清理。

```python
with open("note.txt", "w", encoding="utf-8") as f:
    f.write("hello")
```

这段代码以文本写入方式打开文件并写入`hello`。读取可使用`read()`、`readline()`或遍历文件对象。`"w"`可能清空原内容，修改重要文件前应确认模式并做好备份。

## 十一、C与Python基本区别
<!-- exam: A | 单选、判断、程序阅读 -->

| 项目 | C语言 | Python 3 |
|---|---|---|
| 代码块 | 大括号 | 缩进 |
| 语句结尾 | 多数使用分号 | 通常不需要分号 |
| 逻辑运算 | `&& || !` | `and or not` |
| 整除 | 整数 `/` | `//` |
| 输入 | `scanf` | `input` |
| 输出 | `printf` | `print` |
| 变量类型 | 通常先声明 | 赋值时绑定对象 |
| 常用序列 | 数组、字符数组 | 列表、元组、字符串 |

相同点是都能表示顺序、选择、循环和函数；不同点主要体现在语法形式、类型处理和常用数据结构。

## 十二、高频判断纠错
<!-- exam: A | 判断 -->

1. “程序和算法完全相同。”错误。算法是步骤，程序是具体语言实现。
2. “算法可以无限执行而不结束。”错误。有穷性是算法的基本性质之一。
3. “C和Python标识符都可以数字开头。”错误。
4. “赋值运算符与相等比较都写作 `=`。”错误。比较相等写 `==`。
5. “Python的 `input()` 默认返回整数。”错误。它返回字符串，需要时显式转换。
6. “while循环至少执行一次。”错误。条件初始为假时可以一次不执行。
7. “break会结束整个程序。”错误。它通常只结束最近一层循环。
8. “Python的 `range(5)` 包含5。”错误。它产生0到4。
9. “长度为5的数组有效下标是1到5。”错误。常见下标为0到4。
10. “C字符串不需要结束标志。”错误。C字符串通常以 `\0` 结束。
11. “函数形参就是调用时传入的实参。”错误。形参在定义中，实参在调用处。
12. “列表和元组都可以随意修改。”错误。Python元组通常不可修改。
13. “字典通过连续整数下标访问所有值。”错误。字典主要按键访问。
14. “编译和解释在所有语言实现中完全互斥。”错误。现代实现可能结合多种方式。
15. “指针保存的就是普通数据值，与地址无关。”错误。指针用于保存特定类型对象的地址。
16. “结构体的所有成员必须是同一种类型。”错误。结构体可组合不同类型成员。
17. “以`w`模式打开已有文本文件一定保留原内容。”错误，通常会清空原内容后写入。
18. “Python发生异常后`finally`一定不执行。”错误，`finally`通常无论是否发生异常都会执行。
19. “使用`with open`后仍必须在块外手工关闭文件。”错误，`with`会负责退出时关闭。
