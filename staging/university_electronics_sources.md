# 四门电子通信课程：高校公开资料与题型提炼说明

## 建设口径

本批题库用于大学期末复习，覆盖“电路分析与电工技术、模拟电子技术、数字电子技术、通信原理与高频电子线路”四门课。每门15题，共60题；题型均为9道单选、3道多选、3道判断。

公开资料只用于确认课程边界、期末题型和典型任务。本批题目重新设置了电路结构、条件、数据、选项和讲解，不逐题搬运公开试卷，也不复刻整套试卷。每题通过 `source_url`、`source_title`、`source_chapter` 和 `adaptation_note` 保留可追溯依据。

题目重点不是孤立名词，而是大学期末常见的四类能力：

- 计算：电阻等效、相量与功率、静态工作点、增益、数制与转换、香农容量和调制带宽等。
- 波形：RC暂态、放大器削顶、比较器翻转、触发器边沿采样、数字基带与调制波形等。
- 状态：二极管/三极管工作区、触发器次态、计数器有效与无效状态、通信工作方式等。
- 设计：自锁控制、运放比例/加法器、MUX实现逻辑、存储器扩展、调制—混频—检波链路等。

## 1. 电路分析与电工技术

### 公开来源

1. [浙江万里学院《电路分析及实践》课程教学大纲](https://kcsz.zwu.edu.cn/b2/fb/c6537a176891/page.htm)
   - 明确期末采用选择与综合题。
   - 给出内容比例：基本概念和定律12%，一般分析方法38%，正弦稳态22%，三相10%，动态电路18%。
   - 由此确定题库不能只停留在串并联，还必须覆盖电路定理、交流、三相和动态响应。

2. [南昌航空大学《电路分析》考试大纲（2026年）](https://cgxy.nchu.edu.cn/tzgg/content_184623)
   - 题型包含选择、简单计算和计算题。
   - 范围涵盖KCL/KVL、受控源、Y-△、节点/回路法、叠加与戴维宁、一阶动态、相量、功率、谐振和三相。
   - 据此强化公式的适用条件与完整计算步骤。

3. [福建理工大学《电路》历年试卷资源页](https://ceep.fjut.edu.cn/_t1273/3851/list.htm)
   - 官方页面列出多学年的电路（1）、电路（2）期末试卷及答案资源。
   - 用于确认历年卷与综合应用训练的存在，不从受限附件批量复制题目。

4. [湖南工程学院《电工电子技术》课程教学与考核说明](https://jwc.hnie.edu.cn/swjh/info/1041/1148.htm)
   - 明确直流、正弦、三相、电机与电器控制等工程应用考核范围。
   - 据此补入接触器自锁控制设计题，避免“电路分析”完全替代“电工技术”。

5. [郑州大学《可靠性工程》课程教学大纲](https://gs.zzu.edu.cn/085236jxdg.pdf)
   - 明确包含串联、并联、表决与储备系统的可靠性模型、分析及优化。
   - 仅用于用户近期考试中“两个可靠度0.9元件串联”这一跨课程考点，避免用普通电路试卷目录为可靠性公式作弱关联背书。

### 提炼后的覆盖

电阻等效、KCL、电源变换、戴维宁、容抗、RLC谐振、三相功率、一阶RC波形、可靠性、交流功率补偿、控制回路设计、叠加定理边界、换路定则和最大功率传输。

## 2. 模拟电子技术

### 公开来源

1. [东南大学《模拟电子技术》教学大纲](https://ee.seu.edu.cn/2015/1214/c13622a137739/pagem.htm)
   - 课程定位为技术基础课，强调基本理论、基本知识、基本技能和工程分析能力。
   - 教材采用童诗白、华成英体系，期末考试占80%。

2. [河北工业大学《低频电子线路（模拟电子线路）》课程大纲](https://eie.hebut.edu.cn/yjspy/ssyjs/qrzssyjs/30566.htm)
   - 明确二极管、BJT、FET、三种基本组态、差分电路、集成运放、负反馈及典型应用。
   - 强调交直流参数、小信号模型、性能特点和分析方法。

3. [中南大学模拟电子技术A样卷资源页（第二套）](https://dgdz.csu.edu.cn/theory_teach/simulate_a/testlib/chapter2.html)
   - 官方试卷库页面用于确认样卷训练与电路、波形分析形式。
   - 本题库只提炼任务结构，题目参数与电路情境均重新命制。

### 提炼后的覆盖

二极管分段模型、BJT截止/放大/饱和判定、共射静态与小信号分析、运放比例与加法设计、负反馈、整流滤波、乙类功放、比较器波形、Q点和削顶失真，以及“虚短/虚断”的使用条件。

## 4. 通信原理与高频电子线路

### 公开来源

1. [重庆邮电大学《通信原理》考试大纲](https://scie.cqupt.edu.cn/__local/C/C8/D0/FE064DDCA4225A1B8F69D6F9BFE_AAA435D8_2AC54.pdf)
   - 覆盖通信系统、随机过程、信道与噪声、香农公式、模拟调制、数字基带、数字调制、模拟信号数字化、差错控制和同步。
   - 据此把用户提到的香农公式放回完整通信原理框架中。

2. [哈尔滨理工大学《通信原理》考试大纲](https://graduate.hrbust.edu.cn/_upload/article/files/70/5c/d7928778401bbf6b19e503b58349/b0b01787-8260-4bbe-89aa-c22d7ce84a5a.pdf)
   - 明确单选、填空和分析计算题结构；基础理论约30分、综合设计约20分。
   - 要求掌握信息量、性能指标、信道容量、调制、基带、同步等内容。

3. [青岛科技大学《通信原理》公开期终试卷02](https://xk.qust.edu.cn/__local/3/40/14/333F09921BFA26F205B983C4F4E_62E8F737_1F422.pdf?e=.pdf)

   - 本批题目只采用上述任务类型，概率、信号带宽、序列和参数均重新设置。

4. [河北工业大学《高频电子线路》课程教学大纲](https://eie.hebut.edu.cn/zsjy/yjszs/ssyjs/qrzssyjs/14850.htm)

5. [北京交通大学海滨学院教务处《计算机网络技术与应用》等课程教学大纲汇编](https://jwc.czjtu.edu.cn/info/1082/1227.htm)
   - 公开大纲把无线通信、光纤通信、移动通信和蜂窝通信纳入同一课程范围。
   - 用于支撑半双工、波谱、光纤窗口和网络代际综合辨析题的课程边界；各选项按用户近期考试考点重新表述，并不声称原文出自该页。

### 提炼后的覆盖

## 去重与质量控制

- 标识去重：统一使用 `ue-circuit-*`、`ue-analog-*`、`ue-digital-*`、`ue-comm-*`，本文件内60个ID全部唯一。
- 内容去重：同一知识点若重复出现，分别承担“计算、条件辨析、设计或波形”中的不同任务，不做仅替换数字的平行题。
- 答案自检：单选只含一个合法选项字母；多选答案字母均存在且无重复；判断只使用“正确/错误”，选项对象为空。
- 解析自检：`university_electronics.json` 已通过UTF-8 JSON解析，并检查13个必需字段齐全。
