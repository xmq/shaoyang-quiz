# 六门计算机核心课·高校期末改编题来源与题型提炼

整理日期：2026-07-13

## 建设方法

本题库不是网络试卷拼接。处理流程为：

1. 只选高校官网公开的课程大纲、考试大纲、样卷或期末考核资料页；
2. 对照不同学校的考核范围，提炼共同核心、计算任务和综合分析能力；
3. 保留“题型原型”，重新设计数据、场景、选项和解析，不逐题复制原卷；
4. 每门15题，统一为9道单选、3道多选、3道判断；选择题既检查基础概念，也把原来的填空、应用、算法或设计任务压缩为可在刷题系统中作答的状态推演题；
5. 每道题保存 `source_chapter`、`source_url`、`source_title` 与 `adaptation_note`，用于追踪命题依据，而非声称该题原文来自该页面。

## 1. 操作系统原理

### 官方来源

- [上海海洋大学《操作系统原理》教学大纲](https://xxxy.shou.edu.cn/_upload/article/files/17/ea/c7c93f37492390d8151239c49f06/9ad3a1d9-c4ac-48a3-8c5b-410c8ba843f6.pdf)：明确期末闭卷考核，主要题型包括选择、简答和综合题，关注概念、原理及综合分析能力。
- [重庆科技大学《操作系统原理》教学大纲](https://znxy.cqust.edu.cn/__local/0/BE/76/569646F70A461D08E3BDA8E40AA_291C018B_7E855.pdf?e=.pdf)：范围覆盖操作系统特征、进程描述与控制、进程同步、处理机调度与死锁、存储器、设备和文件管理。

### 题型提炼

- 基础辨析：用户态/内核态、进程与线程、状态转换、文件组织、I/O控制。
- 状态推演：时间片轮转甘特图、PV初值和次序、银行家安全序列。
- 计算分析：分页地址转换、页面置换缺页次数、磁盘访问顺序。
- 本次命题重点：调度、PV、页面置换和死锁共占主要比重，避免只考名词记忆。

## 2. 数据库技术

### 官方来源

- [湖北中医药大学《数据库原理》考试大纲及样卷](https://xxy.hbucm.edu.cn/info/1751/12092.htm)：明确考查关系模型、关系代数、规范化、SQL、数据库设计、恢复、并发控制、安全和完整性，并附样卷入口。
- [江苏第二师范学院《数据库原理及应用》教学大纲与期末考核资料目录](https://ce.jssnu.edu.cn/2025/0925/c5306a69053/page.htm)：公开列出教学大纲，以及多个学年的期中、期末试卷、答案和评分标准资料，体现课程考核持续覆盖理论与应用。

### 题型提炼

- SQL应用：连接、分组、聚合、NULL、外连接条件位置。
- 关系理论：闭包与候选码、函数依赖、2NF/3NF/BCNF。
- 设计分析：E-R联系转关系、键和参照完整性、索引选型。
- 系统机制：优先图可串行化、两段锁、WAL与恢复。
- 本次题目用新的学生选课等模式重建语义，没有复制公开样卷中的表结构或数据。

## 3. 数据结构与算法

### 官方来源

- [安徽工业大学《数据结构》考试大纲](https://jxjy.ahut.edu.cn/www/doc/jsj2.pdf)：给出线性结构35分、二叉树30分、图/查找/排序各10分等内容权重，并含选择、判断、填空、应用和算法题结构。
- [东莞理工学院《数据结构》考试大纲](https://cs.dgut.edu.cn/info/1055/3092.htm)：试卷包含填空、选择、判断、解析和程序设计，要求掌握复杂度、线性表、栈队列、树、图、查找、排序及算法实现分析。

### 题型提炼

- 结构手算：循环队列、遍历序列还原、哈夫曼WPL、折半查找、散列探测。
- 图算法：BFS/DFS、生成树、Dijkstra、拓扑与适用条件。
- 排序分析：分区结果、复杂度、稳定性、额外空间。
- 算法素养：时间复杂度、操作前提、边界条件。
- 为适配客观题系统，把公开大纲强调的“应用题/算法题”改为必须推演才能选择的短综合题。

## 4. 计算机组成原理

### 官方来源

- [安徽工业大学《计算机组成原理》考试大纲](https://jxjy.ahut.edu.cn/www/doc/jsj1.pdf)：覆盖系统性能、数据表示与运算、存储层次、指令系统、CPU、总线与I/O，强调基本部件简单设计。
- [湖南工商大学《计算机组成原理》考试大纲](https://sieim.hutb.edu.cn/zsjy3/yjszs2/ksdg2/content_35783)：公开试卷结构含选择、简答、计算和分析题，要求对硬件问题进行计算、分析和部件设计。

### 题型提炼

- 数值运算：补码真值、溢出、CPU执行时间。
- 部件设计：存储芯片字/位扩展、指令操作码、寻址有效地址。
- 存储综合：Cache地址分解、AMAT、映射与写策略。
- CPU综合：理想流水线时间、三类冒险、RISC/CISC、DMA。
- 题目刻意区分TLB未命中、Cache未命中与缺页，补足容易混淆的层次关系。

## 5. 计算机网络

### 官方来源

- [湖北中医药大学《计算机网络》考试大纲](https://xxy.hbucm.edu.cn/info/1751/12012.htm)：闭卷笔试，系统覆盖体系结构、性能、物理层、链路层、网络层、运输层与应用层，明确奈奎斯特、香农等计算和典型协议分析。
- [广东工商职业技术大学《计算机网络》课程考试大纲](https://zsb.gdbtu.edu.cn/contents/1233/19165.html)：公开名词解释、单选、填空、问答、实验分析等题型，并给出地址规划、代理接入等网络配置分析示例。

### 题型提炼

- 性能与通信：发送时延、奈奎斯特速率。
- 地址规划：CIDR网络地址、主机容量、VLSM思想、最长前缀匹配。
- 报文分析：IPv4分片、TCP SEQ/ACK、三次握手。
- 设备与应用：交换机自学习、ARP、DNS/SMTP/HTTPS/DHCP。
- 协议边界：UDP校验和不等于可靠交付、TIME_WAIT、TCP字节流无消息边界。

## 6. 软件工程

### 官方来源

- [衡阳师范学院《软件工程》教学与考试大纲](https://jkxy.hynu.edu.cn/info/1012/1143.htm)：覆盖软件过程、结构化方法、面向对象与UML、测试、维护、项目管理、风险、质量保证与配置管理，并明确本科课程考试对象和目的。
- [重庆科技大学《软件工程》教学大纲](https://znxy.cqust.edu.cn/info/1155/5636.htm)：强调软件过程、需求、设计、测试和开发管理的实践运用，期末采用笔试评价课程目标。

### 题型提炼

- 场景选择：过程模型、维护类型、体系结构或设计原则。
- 需求与建模：可验证需求、DFD平衡、UML图用途、内聚耦合。
- 测试设计：边界值、完整测试用例、控制流图与环形复杂度。
- 项目管理：关键路径、风险应对、团队增员的沟通代价。
- 判断题专门纠正“敏捷不要文档”“测试可证明无错”“增员一定赶工”等绝对化说法。
