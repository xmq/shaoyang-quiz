# Office 与 WPS · 三色回忆卡

> 先回答功能和对象，再揭示答案；版本差异按题干环境判断。

## 一、组件与文件基础

### 回忆卡 1｜三类组件

先答：Word/WPS文字、Excel/WPS表格、PowerPoint/WPS演示分别解决什么问题？

- 🔴 文字处理文档和页面；表格处理数据、计算和图表；演示组织幻灯片并放映。
- 🔵 简答按“组件名称—基本对象—主要作用”写。
- ⚫ 三者可交换对象，但复制后格式、链接和编辑能力可能变化。

### 回忆卡 2｜格式与保存

先答：三类现代默认、旧版和含宏格式是什么？保存与另存为有何区别？

- 🔴 现代常用格式为`.docx/.xlsx/.pptx`，含宏常用`.docm/.xlsm/.pptm`；保存更新当前文件，另存为可改名、位置或格式。
- 🔵 交付前检查文件名、位置、格式、能否重新打开和是否保留编辑结构。
- ⚫ PDF主要保持版式；自动恢复和云同步不能代替主动保存与备份。

### 回忆卡 3｜Office与WPS差异

先答：两套软件哪些地方相近，哪些内容最需要兼容检查？

- 🔴 基本对象和操作思路相近；字体、分页、宏、公式、图表和动画需复核。
- 🔵 答操作题先写功能目标和对象，再写指定软件的功能入口。
- ⚫ 菜单位置和默认值随版本变化，不能把某版界面当成绝对规则。

## 二、Word/WPS文字

### 回忆卡 1｜编辑、字符与段落

先答：字符格式和段落格式各控制什么？剪切、复制和格式刷有何区别？

- 🔴 字符格式管字体字号等，段落格式管对齐、缩进和间距；剪切移动内容，复制保留原内容，格式刷只复制格式。
- 🔵 判断题先看作用对象是文字本身还是整段排版。
- ⚫ `Backspace/Delete` 在有选区时都可清除所选内容，不能只背左右删除。

### 回忆卡 2｜样式、分页与分节

先答：样式有什么作用？分页符和分节符最关键的区别是什么？

- 🔴 样式统一格式并支持大纲和目录；分页符只换页，分节符建立可独立设置方向、页码和页眉页脚的节。
- 🔵 不同页眉页脚：分节 → 取消“链接到前一节” → 分别设置。
- ⚫ 手工加粗放大不等于标题样式；分页符不能代替分节符。

### 回忆卡 3｜图片、表格与长文档

先答：嵌入型与浮动型图片怎样区别？自动目录和邮件合并依赖什么？

- 🔴 嵌入型图片像字符排版，浮动型可环绕；目录依赖标题样式；邮件合并由主文档、数据源和合并域组成。
- 🔵 长文档按“样式 → 分节 → 题注/引用 → 目录 → 更新域”处理。
- ⚫ 表格中按Delete通常只清内容；删除批注不等于接受修订。

## 三、Excel/WPS表格

### 回忆卡 1｜对象、输入与引用

先答：工作簿、工作表、单元格是什么层级？四种A1引用怎样变化？

- 🔴 工作簿含工作表，单元格是行列交点；`A1`行列都变，`$A$1`都锁，`$A1/A$1`分别锁列/行。
- 🔵 填充公式前先判断哪些量变化、哪些量固定，再放`$`。
- ⚫ 学号等数字标识常应按文本输入；显示格式不等于改变底层值。

### 回忆卡 2｜函数与错误值

先答：SUM、AVERAGE、COUNT、COUNTA、IF分别做什么？常见错误值表示什么？

- 🔴 前四者依次求和、平均、数值个数、非空个数；IF按条件返回结果；`#DIV/0!/#VALUE!/#REF!`常指除零、类型不合、引用失效。
- 🔵 函数题先认功能，再核对区域、条件、括号和引用。
- ⚫ COUNT不统计普通文本；`######`常因列宽，不代表数据一定丢失。

### 回忆卡 3｜精确与近似查找

先答：VLOOKUP和XLOOKUP怎样选择精确/近似匹配？

- 🔴 VLOOKUP第四参数`FALSE/0`是精确匹配，`TRUE/1`或省略通常是近似匹配且查找列应升序；XLOOKUP在支持的现代版本中默认精确匹配。
- 🔵 唯一编号查记录用精确匹配；按已排序区间下限查等级才考虑近似匹配。
- ⚫ 近似匹配未排序可能返回错误值；XLOOKUP支持情况随Office/WPS版本而异。

### 回忆卡 4｜数据管理、图表与打印

先答：排序、筛选、条件格式和数据验证分别改变什么？四类常用图表怎样选？

- 🔴 排序重排记录，筛选隐藏记录，条件格式改显示，数据验证限输入；比较用柱形、趋势用折线、构成用饼图、数值关系用散点。
- 🔵 打印检查区域、方向、页边距、缩放、分页、标题和预览。
- ⚫ 筛选不删除数据；冻结窗格不是保护，条件格式也不是数据验证。

### 回忆卡 5｜保护与加密

先答：单元格锁定、工作表保护、工作簿结构保护和文件加密分别保护什么？

- 🔴 锁定在启用工作表保护后限制改单元格；工作表保护防误改内容；工作簿保护限制表结构；文件加密控制能否打开文件。
- 🔵 先判断要防“改单元格、改工作表结构还是未授权打开”，再选功能。
- ⚫ 保护工作表不等于文件加密；锁定未配合工作表保护通常不生效，加密也不能代替备份。

## 四、PowerPoint/WPS演示

### 回忆卡 1｜视图与结构

先答：普通、浏览、备注、放映视图各用于什么？主题、版式、母版如何分工？

- 🔴 普通编辑、浏览调序、备注写讲稿、放映全屏；主题管颜色字体，版式管占位符，母版统一多个版式。
- 🔵 全局修改优先进入母版，不逐页重复设置。
- ⚫ 占位符属于版式，文本框是独立对象；直接格式可能覆盖母版默认值。

### 回忆卡 2｜切换、动画与链接

先答：切换、动画、超链接和动作按钮分别作用在哪里？

- 🔴 切换作用于幻灯片之间，动画作用于页内对象，链接和动作按钮用于跳转。
- 🔵 动画核对对象、效果、开始方式、顺序和持续时间。
- ⚫ 转场和动画越多不代表演示越好，二者作用对象不能混淆。

### 回忆卡 3｜放映与表达

先答：排练计时、录制放映和演讲者视图有什么作用？

- 🔴 排练记录换片时间，录制保存旁白/计时等，演讲者视图显示备注、下一页和时间。
- 🔵 放映前查字体、媒体路径、链接、动画顺序和目标设备兼容。
- ⚫ 备注供演讲者使用，不应把全部讲稿堆入幻灯片正文。

## 五、基础答题与判断纠错

### 回忆卡 1｜概念区别题

先答：比较两个办公概念时，怎样写出可得分的区别？

- 🔴 先分别定义，再写作用对象、主要作用和一项关键区别。
- 🔵 使用“前者……；后者……；因此……”的对照句式。
- ⚫ 只写“功能不同、用法不同”没有给出实质区别。

### 回忆卡 2｜操作步骤题

先答：不确定具体版本菜单时，怎样写操作步骤？

- 🔴 选定对象 → 打开对应功能 → 设置参数 → 确认应用范围 → 检查结果。
- 🔵 Word辨字符/段落/节，Excel辨单元格/公式/数据，PPT辨单页或全局版式。
- ⚫ 只背图标位置容易受Office/WPS和版本变化影响。

### 回忆卡 3｜高频绝对化陷阱

先答：为什么“WPS与Office完全相同、筛选会删除、切换作用于图片”都错？

- 🔴 两套软件存在版本差异；筛选只隐藏记录；切换作用于幻灯片之间，图片效果属于动画。
- 🔵 判断题遇到“所有、永远、完全”先核对对象、条件和版本。
- ⚫ 绝对化词只是风险信号，不能脱离知识点机械判错。
