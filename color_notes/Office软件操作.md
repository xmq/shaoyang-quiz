# Microsoft Office + WPS 办公软件操作·期末复习三色笔记

> 只用于考前速记：🔴核心必背；🔵公式、步骤、快捷键与踩分词；⚫条件、版本差异与判断陷阱。Microsoft Office 与 WPS Office 采用共性主线；题干指定软件、版本或操作系统时，以指定环境为准。

## 一、Office 共性基础

- 🔴 组件对应：Microsoft Word ↔ WPS 文字，Microsoft Excel ↔ WPS 表格，Microsoft PowerPoint ↔ WPS 演示；核心对象和操作逻辑相近，但界面入口和特色功能不必完全相同。
- 🔴 两套软件均可处理常见 `.docx/.xlsx/.pptx` 并导出 PDF；复杂宏、域、字体、动画、图表、公式和插件跨软件时可能不完全兼容。
- 🔴 Office 文档的基本操作是新建、打开、保存、另存为、关闭、打印和导出；“保存”更新当前文件，“另存为”可改变位置、名称或格式。
- 🔴 Word、Excel、PowerPoint 的现代默认格式分别是 `.docx`、`.xlsx`、`.pptx`；旧格式分别常见 `.doc`、`.xls`、`.ppt`。
- 🔴 含 VBA 宏的现代格式常见 `.docm`、`.xlsm`、`.pptm`；模板常见 `.dotx/.dotm`、`.xltx/.xltm`、`.potx/.potm`。
- 🔵 高频快捷键：`Ctrl+N` 新建，`Ctrl+O` 打开，`Ctrl+S` 保存，`F12` 常用于另存为，`Ctrl+P` 打印，`Ctrl+Z/Y` 撤销/恢复，`Ctrl+X/C/V` 剪切/复制/粘贴，`Ctrl+A` 全选，`Ctrl+F/H` 查找/替换。
- 🔵 跨软件答题：写功能目标和对象 → 写共性步骤 → 按题干替换为 Word/Excel/PowerPoint 或 WPS 文字/表格/演示的菜单名称 → 检查兼容与输出。
- ⚫ 自动保存、自动恢复与手动保存不是同一概念；恢复文件不能代替规范保存和备份。
- ⚫ PDF 通常用于保持版式和分发，不等同于可完整保留源文档所有编辑结构；兼容模式可能限制新功能。
- ⚫ Microsoft 365、Office 2016/2019/2021 与 WPS 个人版/教育版/专业版的默认值、云协作、宏和高级功能存在差异；不能把某版按钮位置写成绝对结论。

## 二、Word：编辑与格式

- 🔴 字符格式包括字体、字号、字形、颜色、字符间距和文字效果；段落格式包括对齐、缩进、段前段后、行距、制表位和项目符号/编号。
- 🔴 `Backspace` 通常删插入点左侧字符，`Delete` 通常删右侧字符；文本已选中时二者均可删除所选内容。
- 🔴 格式刷复制格式而非正文；单击通常使用一次，双击可连续应用，按 `Esc` 退出。
- 🔵 常用快捷键：`Ctrl+B/I/U` 加粗/倾斜/下划线，`Ctrl+L/E/R/J` 左/居中/右/两端对齐，`Ctrl+Enter` 插入分页符，`Shift+Enter` 插入手动换行符。
- 🔵 清除直接格式后可回到样式定义；统一长文档应优先使用“样式”，而非逐段手工改字体。
- ⚫ 中文字号中“初号”大于“一号”，“二号”大于“四号”；以磅为单位时数值越大字体越大，两套序列不能按数字直接比较。
- ⚫ 固定值行距可能遮挡过大的文字；多倍行距会随字体等因素计算。

## 三、Word：页面、节与页眉页脚

- 🔴 分页符只强制从下一页开始；分节符建立新“节”，不同节可分别设置纸张方向、页边距、分栏、页码格式和页眉页脚。
- 🔴 分节符常见“下一页、连续、奇数页、偶数页”；“连续”分节可在同页改变分栏等设置。
- 🔵 设置不同页眉页脚：插入分节符 → 进入后一节页眉/页脚 → 关闭“链接到前一节” → 分别编辑 → 必要时设置页码起始值。
- 🔵 只需同一节“首页不同”或“奇偶页不同”时，勾选相应选项即可，不必一定新增分节符。
- 🔴 页面设置高频项：纸张大小、方向、页边距、装订线、文字方向、分栏和应用范围。
- ⚫ 删除分节符会使相邻节合并，并可能改变前一部分版式；分页符不能代替分节符。

## 四、Word：对象、表格与长文档

- 🔴 嵌入型图片按字符参与排版；四周型、紧密型、衬于文字下方、浮于文字上方等属于浮动环绕。
- 🔴 组合对象通常要求对象为浮动型；图片的“裁剪”改变显示区域，不等于删除源图像数据。
- 🔴 表格中按 `Delete` 通常只清除内容；删除单元格、行、列或整表才改变结构。
- 🔵 Word 表格可用 `=SUM(ABOVE)`、`=AVERAGE(LEFT)` 等公式；源数据改变后常需选中结果按 `F9` 更新域。
- 🔴 标题样式是自动目录、大纲导航和多级编号的基础；手工加粗放大不等于应用标题样式。
- 🔵 长文档流程：定义样式 → 应用标题层级 → 插入分页/分节 → 设置题注和交叉引用 → 插入目录 → 全选后更新域。
- 🔴 邮件合并由主文档、数据源和合并域组成，可批量生成信函、标签、证书等。
- 🔴 修订记录正文增删和格式变化；批注用于讨论。删除批注不等于接受修订。
- ⚫ 自动目录内容或页码变化后要“更新目录”；不要手工逐条改目录。

## 五、Excel：对象、输入与引用

- 🔴 工作簿是文件，工作表是工作簿内的表页；单元格地址先列后行，如 `B3`。
- 🔴 Excel 2007 及以后每表通常为 1,048,576 行、16,384 列，末列 `XFD`；旧版 Excel 2003 为 65,536 行、256 列。
- 🔴 公式以 `=` 开头；长学号、身份证号等应按文本输入，可先设文本格式或在前面输入英文单引号。
- 🔴 相对引用 `A1` 随复制改变；绝对引用 `$A$1` 行列均锁定；混合引用 `$A1` 锁列、`A$1` 锁行。
- 🔵 编辑公式时 `F4` 可循环切换引用形式；跨表引用示例：`'期末成绩'!$B$2`。
- 🔵 填充公式前先问两件事：哪个量随位置变化，哪个量必须锁定；再决定 `$` 放在行、列还是二者。
- ⚫ `######` 最常见于列宽不足，也可能因负日期/时间等；不能一概判定数据丢失。
- ⚫ 工作表默认数量和可添加数量随版本、设置与内存变化，“永远是 3 张”“最多 255 张”均不严谨。

## 六、Excel：公式、函数与错误值

- 🔴 运算顺序以括号最可靠；常见顺序为引用运算、负号/百分号、乘方、乘除、加减、连接 `&`、比较。
- 🔵 基本统计：`SUM` 求和，`AVERAGE` 平均，`MAX/MIN` 极值，`COUNT` 数值个数，`COUNTA` 非空个数。
- 🔵 条件函数：`IF(条件,真值,假值)`；`COUNTIF/COUNTIFS` 条件计数；`SUMIF/SUMIFS` 条件求和；`AVERAGEIF(S)` 条件平均。
- 🔵 常考函数：`ROUND` 四舍五入，`LEFT/RIGHT/MID` 截取文本，`LEN` 字符数，`TODAY/NOW` 日期时间，`RANK.EQ` 排名，`VLOOKUP/XLOOKUP` 查找。
- 🔵 多条件判断常用 `AND`、`OR` 嵌入 `IF`；文本条件要加英文双引号，如 `">=60"`。
- 🔵 查找踩分：明确“查找值、查找区域、返回列/数组、精确或近似匹配”；传统 `VLOOKUP` 精确匹配最后参数写 `FALSE` 或 `0`。
- 🔴 常见错误：`#DIV/0!` 除零，`#VALUE!` 类型不合，`#NAME?` 名称无法识别，`#REF!` 引用失效，`#N/A` 无可用匹配，`#SPILL!` 动态数组溢出区受阻。
- ⚫ `COUNT` 不统计普通文本；`COUNTIF` 条件区域与 `SUMIF` 求和区域含义不同；函数名称或功能可能随版本新增。

## 七、Excel：数据管理、图表与打印

- 🔴 排序可按多个关键字；必须扩展选择整条记录，否则会破坏同行字段对应关系。
- 🔴 筛选只是暂时隐藏不符合条件的记录，不删除数据；高级筛选可使用条件区域并把结果复制到其他位置。
- 🔴 传统分类汇总前先按分类字段排序；数据透视表用于按字段分组、汇总和交叉分析，源数据变化后通常要刷新。
- 🔴 数据验证可限制类型、范围或提供下拉列表；条件格式按规则改变显示格式，二者均不等于加密保护。
- 🔵 图表选择：类别比较用柱形/条形；时间趋势用折线；少量类别构成用饼图；两个数值变量关系用散点图。
- 🔵 图表基本元素：图表标题、坐标轴、图例、数据系列、数据标签、网格线；答操作题时先选数据区域再插入合适图表。
- 🔴 冻结窗格用于滚动时保持标题可见，不是禁止编辑；锁定单元格只有在保护工作表后才发挥限制作用。
- 🔵 打印检查：打印区域 → 纸张方向/页边距 → 缩放 → 分页 → 重复标题行/列 → 页眉页脚 → 打印预览。
- ⚫ 普通图表通常随既有源区域数据变化而更新，但新增记录若不在引用区域内，未必自动纳入。

## 八、PowerPoint：结构、视图与母版

- 🔴 普通视图用于编辑；幻灯片浏览视图适合调整顺序和整体检查；备注页用于编辑演讲者备注；阅读视图在窗口中放映。
- 🔴 主题控制整体颜色、字体和效果；版式规定占位符结构；母版统一控制多个版式及其默认格式。
- 🔵 全局统一格式：视图 → 幻灯片母版 → 修改母版/相应版式 → 关闭母版视图；不要逐页手工修改。
- 🔴 新建幻灯片常继承当前版式；复制幻灯片可选择使用目标主题或保留源格式。
- 🔴 占位符属于版式结构，可容纳文本或内容；文本框是单独插入的对象，二者不能完全等同。
- ⚫ 修改母版不会必然覆盖幻灯片上已直接设置的全部格式；直接格式可能优先显示。

## 九、PowerPoint：对象、动画与放映

- 🔴 幻灯片切换作用于页面之间；动画作用于页面内对象。一个对象可设置多个动画，但要管理顺序和触发方式。
- 🔴 动画类型通常分进入、强调、退出、动作路径；开始方式常见“单击时、与上一动画同时、上一动画之后”。
- 🔵 动画设置流程：选对象 → 添加动画 → 设置效果选项 → 调整动画窗格顺序 → 设置开始/持续时间/延迟 → 预览。
- 🔴 超链接可跳转到网页、文件、邮件地址或指定幻灯片；动作按钮还可在单击/鼠标移过时执行跳转或声音。
- 🔵 `F5` 从头放映，`Shift+F5` 从当前页放映，`Esc` 退出；排练计时记录每页时间，录制可包含旁白、墨迹和计时。
- 🔴 插入音视频后应检查文件是否嵌入、兼容、可播放以及放映起止方式；“压缩媒体”可减小文件但可能损失质量。
- 🔴 讲义母版控制打印讲义版式；备注母版控制备注页，不要与幻灯片母版混淆。
- 🔵 演示设计踩分词：一页一重点、层级清楚、对比充分、字体可读、图表服务结论、动画克制、来源标注。
- ⚫ 动画和转场越多不等于效果越好；自动换片与单击换片可同时设置，实际行为取决于放映设置。

## 十、综合操作题答题法

- 🔵 Word：先辨“字符/段落/页面/节/对象/样式”，再选命令；涉及不同页设置优先想到“分节”。
- 🔵 Excel：先写目标 → 确定数据区域 → 写公式 → 判断引用是否锁定 → 填充 → 检查边界值和错误值。
- 🔵 PowerPoint：先辨“单页对象”还是“全局版式”；全局改外观优先考虑主题/母版，页面间效果用切换，页内对象效果用动画。
- 🔵 操作题踩分表述：选定对象 → 打开对应选项卡/对话框 → 设置具体参数 → 确定/应用 → 检查应用范围。
- ⚫ 菜单位置会随版本和 WPS/Microsoft Office 改变；期末主观题优先写“功能名称和操作逻辑”，不要只背图标位置。
