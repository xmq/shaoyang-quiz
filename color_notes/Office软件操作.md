# Microsoft Office + WPS · 三色回忆卡

> 先答功能和对象，再核对操作。Word/WPS文字、Excel/WPS表格、PowerPoint/WPS演示采用共性主线；题干指定软件、版本或系统时以指定环境为准。

## 一、文件、格式与兼容

### 回忆卡 1｜组件与文件格式

先答：三组对应组件及现代默认、旧版、含宏格式分别是什么？

- 🔴 Word/文字：`.docx/.doc/.docm`；Excel/表格：`.xlsx/.xls/.xlsm`；PowerPoint/演示：`.pptx/.ppt/.pptm`。
- 🔵 先识别文档类型，再判断是否需要保留宏、模板、编辑结构或只导出 PDF。
- ⚫ PDF 便于保持版式，但不完整保留源文档编辑结构。

### 回忆卡 2｜保存与恢复

先答：保存、另存为、自动保存、自动恢复和备份有什么区别？

- 🔴 保存更新当前文件；另存为可改名称、位置或格式；自动保存/恢复用于降低意外损失，不能代替规范保存和备份。
- 🔵 关键节点用 `Ctrl+S`，交付前检查路径、格式、版本和能否重新打开。
- ⚫ 恢复文件可能不含最后修改，云同步也可能同步误删或错误版本。

### 回忆卡 3｜跨软件兼容

先答：Office 与 WPS 打开同一文件时，哪些内容最需要复核？

- 🔴 重点检查字体、分页、域、宏、公式、图表、动画、插件和协作功能。
- 🔵 跨软件答题写“功能目标 → 共性步骤 → 指定软件菜单 → 兼容检查 → 输出验证”。
- ⚫ 两套软件对象逻辑相近，但按钮位置、默认值和高级功能不必完全相同。

## 二、Word/WPS文字：格式与长文档

### 回忆卡 1｜字符、段落与样式

先答：字符格式、段落格式和样式分别控制什么？

- 🔴 字符控制字体字号等；段落控制对齐、缩进、间距等；样式保存成组格式并支持长文档结构化管理。
- 🔵 长文档先定义并应用标题/正文样式，不逐段手工修改。
- ⚫ 手工加粗放大不等于标题样式，不能可靠生成自动目录。

### 回忆卡 2｜分页与分节

先答：分页符和分节符的最小区别是什么？怎样做不同页眉页脚？

- 🔴 分页符只换页；分节符建立可独立设置方向、页边距、分栏、页码和页眉页脚的新节。
- 🔵 插分节符 → 进入后一节页眉页脚 → 取消“链接到前一节” → 分别设置。
- ⚫ 删除分节符可能改变前一部分版式；同一节“首页不同”不一定需要再分节。

### 回忆卡 3｜对象、表格与目录

先答：图片环绕、表格删除、题注和自动目录分别依赖什么对象？

- 🔴 嵌入型图片按字符排版，浮动型可环绕；Delete 常只清表格内容；自动目录依赖标题样式，题注支持交叉引用。
- 🔵 流程：样式层级 → 分页/分节 → 题注/引用 → 插入目录 → 更新全部域。
- ⚫ 裁剪改变显示区域不必然删除源数据；正文变化后目录和域通常要更新。

## 三、Excel/WPS表格：引用与函数

### 回忆卡 1｜工作簿、单元格与输入

先答：工作簿、工作表、单元格是什么层级？长数字怎样避免科学计数或丢位？

- 🔴 工作簿是文件，工作表是其中表页，单元格地址先列后行；学号、身份证等应按文本输入。
- 🔵 输入前判断数据是计算量还是标识符，再设置数值、日期或文本格式。
- ⚫ 显示格式不等于改变底层值；`######` 常因列宽，也可能与日期时间异常有关。

### 回忆卡 2｜相对、绝对与混合引用

先答：复制公式时 `A1`、`$A$1`、`$A1`、`A$1` 哪部分变化？

- 🔴 相对引用行列都变；绝对引用都锁；混合引用只锁带 `$` 的行或列。
- 🔵 先问“哪个量随填充变化，哪个必须固定”，再放 `$`；`F4` 常用于循环切换引用形式。
- ⚫ 跨表引用还要保留表名，含空格的表名通常需要引号。

### 回忆卡 3｜函数、条件与查找

先答：COUNT/COUNTA、IF、条件统计和查找函数分别解决什么任务？

- 🔴 COUNT 数数值，COUNTA 数非空；IF 分支；COUNTIF(S)/SUMIF(S) 条件统计；VLOOKUP/XLOOKUP 按键查找。
- 🔵 写查找时明确“查找值—区域—返回列/数组—精确或近似”；传统 VLOOKUP 精确匹配末参数写 `FALSE/0`。
- ⚫ COUNT 不统计普通文本；函数可用性和名称随版本变化。

## 四、Excel/WPS表格：数据分析与输出

### 回忆卡 1｜排序、筛选与汇总

先答：排序、筛选、分类汇总和数据透视表分别改变什么？

- 🔴 排序重排记录；筛选隐藏不符合条件的行；传统分类汇总按分类字段分组统计；透视表按字段重组汇总。
- 🔵 排序要扩展整条记录；分类汇总前先排序；源数据更新后刷新透视表。
- ⚫ 筛选不删除数据，只排一列可能破坏同行字段对应关系。

### 回忆卡 2｜图表选择

先答：类别比较、时间趋势、构成和两个数值变量关系分别选什么图？

- 🔴 柱形/条形看比较，折线看趋势，少量类别饼图看构成，散点图看数值关系。
- 🔵 先选正确数据区域，再检查标题、坐标轴、系列、图例和数据标签。
- ⚫ 图表服务结论而非装饰；同一数据并非只能使用一种图表。

### 回忆卡 3｜验证、保护与打印

先答：数据验证、条件格式、冻结窗格、锁定单元格分别做什么？打印前查什么？

- 🔴 验证限制输入，条件格式按规则改变显示，冻结保持标题可见；锁定须配合保护工作表才限制编辑。
- 🔵 打印检查：区域 → 方向/页边距 → 缩放 → 分页 → 重复标题 → 页眉页脚 → 预览。
- ⚫ 条件格式不是数据校验，冻结窗格也不是禁止编辑。

## 五、PowerPoint/WPS演示

### 回忆卡 1｜主题、版式与母版

先答：主题、版式、母版、占位符和文本框如何分工？

- 🔴 主题定颜色字体效果；版式定占位符结构；母版统一控制多个版式；占位符属于版式，文本框是独立对象。
- 🔵 全局修改走“视图 → 幻灯片母版 → 修改母版/版式 → 关闭母版”。
- ⚫ 直接设置的单页格式可能覆盖母版默认效果。

### 回忆卡 2｜切换、动画与交互

先答：切换、动画、超链接和动作按钮分别作用在哪里？

- 🔴 切换作用于幻灯片之间；动画作用于页内对象；超链接/动作按钮跳转到页面、文件或网址。
- 🔵 多对象动画检查对象、效果、开始方式、顺序、持续时间和触发条件。
- ⚫ 动画和转场越多不等于表达更好；自动与单击换片行为取决于放映设置。

### 回忆卡 3｜放映与演示设计

先答：排练计时、录制放映和演讲者视图各提供什么？

- 🔴 排练记录换片时间；录制可保存旁白、墨迹和计时；演讲者视图帮助查看备注和下一页。
- 🔵 设计检查：一页一重点、层级清楚、字体可读、图表支持结论、来源可追溯。
- ⚫ 备注供演讲者使用，不应把全部讲稿堆在幻灯片正文。

## 六、综合操作与故障判断

### 回忆卡 1｜先辨对象再找命令

先答：面对操作题，为什么第一步不是背菜单位置？

- 🔴 应先确认对象和目标：Word 看字符/段落/页面/节/对象，Excel 看单元格/区域/公式/数据，PPT 看单页对象或全局版式。
- 🔵 通用答法：选定对象 → 打开相应功能 → 设置参数 → 确认应用范围 → 检查结果。
- ⚫ 只背图标位置容易受 Office/WPS 和版本差异影响。

### 回忆卡 2｜常见故障定位

先答：排版错位、公式错误、放映异常应怎样逐层排查？

- 🔴 先复现，再查对象选择、格式/公式、引用或链接、版本兼容和输出设置。
- 🔵 Word 查样式/节/环绕；Excel 查类型/引用/错误值；PPT 查母版/动画顺序/媒体路径。
- ⚫ 不要用“全部重做”代替定位，修改后要重新打开、打印预览或放映验证。

### 回忆卡 3｜交付前检查

先答：一份办公文件交付前至少检查哪些项目？

- 🔴 内容正确、格式一致、链接/公式更新、隐私清理、字体媒体兼容、文件名与输出格式正确。
- 🔵 保存可编辑源文件，再按要求导出 PDF、打印稿或演示包，并在目标环境复核。
- ⚫ “在本机正常”不代表目标电脑、WPS或不同 Office 版本一定一致。
