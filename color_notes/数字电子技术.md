# 数字电子技术｜三色回忆卡

## 一、数字信号与逻辑门

### 回忆卡1：数字信号的特点

先答：数字信号有什么特点和优点？

- 🔴 状态离散；抗干扰较强、便于存储处理、易集成、可靠性较高。
- 🔵 简答先定义“离散状态”，再列三项优点。
- ⚫ 逻辑0和1不固定等于0 V和1 V，具体电平由器件决定。

### 回忆卡2：与、或、非门

先答：与门、或门、非门分别在什么条件下输出1？

- 🔴 与门全1出1；或门至少一个1出1；非门把输入取反。
- 🔵 用“同时满足、任一满足、相反”三个词快速判断。
- ⚫ “或”包含输入同时为1的情况，不是只能有一个输入为1。

### 回忆卡3：异或与同或

先答：异或门和同或门的功能有什么区别？

- 🔴 异或在输入不同时出1，同或在输入相同时出1。
- 🔵 异或可记作“不等检测”，同或可记作“相等检测”。
- ⚫ 两输入异或不是普通或门；输入都为1时异或输出0。

### 回忆卡4：二进制与十六进制

先答：一位十六进制对应几位二进制，十进制13怎样表示？

- 🔴 一位十六进制对应四位二进制；13=`1101₂`=`D₁₆`。
- 🔵 十六进制转二进制时逐位换成四位二进制组。
- ⚫ A～F分别表示十进制10～15，不是普通字母变量。

### 回忆卡5：BCD、ASCII与奇偶校验

先答：8421 BCD码、ASCII码和奇偶校验分别有什么作用？

- 🔴 8421 BCD用四位代码表示一位十进制数字；ASCII表示字符；奇偶校验用校验位检查传输差错。
- 🔵 BCD抓“逐个十进制位编码”，ASCII抓“标准7位字符码”，奇偶校验抓“1的总数为奇数或偶数”。
- ⚫ BCD的 `1010`～`1111`无效；字符 `'0'` 不等于数值0；奇偶校验不能可靠发现偶数个比特错误，也不能纠错。

## 二、逻辑代数与组合电路

### 回忆卡6：德摩根定律

先答：德摩根定律的两种基本形式是什么？

- 🔴 `(A+B)̅=A̅·B̅`；`(A·B)̅=A̅+B̅`。
- 🔵 口诀：“去总非，换运算，各项取反”。
- ⚫ 不能只换与、或而漏掉每个变量上的取反。

### 回忆卡7：组合逻辑的特点

先答：组合逻辑电路有什么特点，常见器件有哪些？

- 🔴 输出只由当前输入决定，无记忆；常见编码器、译码器、选择器、加法器、比较器。
- 🔵 简答用“当前输入—无记忆—举例”三步。
- ⚫ 组合电路仍可能因门延迟不同产生竞争冒险。

### 回忆卡8：编码器与译码器

先答：编码器和译码器分别完成什么功能？

- 🔴 编码器把有效输入变成代码；译码器把代码翻译成对应输出。
- 🔵 抓住信息变换方向相反，可分别举键盘编码和地址译码。
- ⚫ 译码器不是把模拟量变成数字量，那是ADC的任务。

### 回忆卡9：半加器与全加器

先答：半加器和全加器的主要区别是什么？

- 🔴 半加器只加两个本位输入；全加器还接收低位进位。
- 🔵 两者都输出本位和与向高位的进位。
- ⚫ “全加”不表示一次完成任意位数加法，多位加法需多个全加器连接。

## 三、触发器与时序电路

### 回忆卡10：组合与时序的区别

先答：组合逻辑和时序逻辑最根本的区别是什么？

- 🔴 时序逻辑输出还与原状态有关并具有记忆，组合逻辑没有。
- 🔵 各举一例：译码器属于组合，计数器属于时序。
- ⚫ 时序电路通常含触发器并受时钟控制，但异步控制端也能直接改变状态。

### 回忆卡11：RS触发器

先答：常见高有效RS触发器的保持、置1、置0和禁用输入是什么？

- 🔴 `00`保持、`10`置1、`01`置0、`11`禁用。
- 🔵 顺序按 `S、R` 读，S对应Set，R对应Reset。
- ⚫ 低有效NAND型输入规则不同，必须先看题图有效电平。

### 回忆卡12：JK触发器

先答：JK触发器四种输入组合分别执行什么功能？

- 🔴 `00`保持、`01`置0、`10`置1、`11`翻转。
- 🔵 与RS对照：JK把原来的禁用组合改成翻转。
- ⚫ “RJ触发器”不是规范通用名称，需结合状态表判断题意。

### 回忆卡13：D和T触发器

先答：D触发器和T触发器的下一状态规则是什么？

- 🔴 D：`Q下一=D`；T：0保持、1翻转。
- 🔵 波形题先找有效时钟沿，再读取该时刻输入。
- ⚫ 两个有效沿之间输入变化通常不会立即改变边沿触发器的状态。

## 四、寄存器、计数器与转换器

### 回忆卡14：寄存器与计数器

先答：寄存器和计数器分别有什么作用？

- 🔴 寄存器暂存二进制数据；计数器对输入脉冲计数。
- 🔵 移位寄存器还可用于移位、串并转换；计数器可用于分频和定时。
- ⚫ 模N计数器有N个有效状态，不一定用满全部二进制组合。

### 回忆卡15：RAM与ROM

先答：RAM和ROM的基本区别是什么？

- 🔴 RAM可随机读写且通常易失；ROM以读取为主且非易失。
- 🔵 从“读写能力—断电保持—用途”三个角度对比。
- ⚫ RAM不是“只能读”，ROM也不等于所有类型永远不能改写。

### 回忆卡16：ADC与DAC

先答：ADC和DAC分别完成什么转换？

- 🔴 ADC模拟转数字；DAC数字转模拟。
- 🔵 ADC常见过程为采样、保持、量化、编码。
- ⚫ ADC位数越多通常分辨率越高，但速度和成本还取决于结构。

### 回忆卡17：555定时器三类功能

先答：555定时器的单稳态、无稳态和施密特触发分别有什么特点与作用？

- 🔴 单稳态受触发后输出一定宽度脉冲；无稳态自行连续振荡；施密特触发利用两个阈值整形并抗干扰。
- 🔵 用途依次记为“定时延时—时钟脉冲—波形整形”。
- ⚫ 无稳态不需要每个周期都由外部触发；施密特触发的两个阈值形成回差，不是同一个阈值。

### 回忆卡18：逻辑场景先问什么

先答：拿到门禁、报警或计数场景，怎样判断用组合逻辑还是时序逻辑？

- 🔴 只看当前输入用组合逻辑；需要记住过去状态、保存或计数用时序逻辑。
- 🔵 “任一路报警”对应或；“全部条件满足”对应与；“恰好不同”常对应异或；保持报警需触发器。
- ⚫ 纯组合逻辑没有记忆；若要求传感器恢复后仍报警，必须增加状态保存和复位机制。

### 回忆卡19：模N计数器最少触发器

先答：模10计数器至少需要几个触发器，为什么？

- 🔴 4个，因为满足 `2^n≥10` 的最小整数是4。
- 🔵 `2³=8<10≤16=2⁴`，所以3个状态不够、4个足够。
- ⚫ 4个触发器共有16种编码，模10只用10种；还要考虑非法状态的处理，不能说它自然就是模16。

## 五、微控制器、接口与调试

### 回忆卡20：微控制器最小系统

先答：单片机上电无反应，先检查哪几个基本条件？

- 🔴 按电源与去耦→复位→时钟→程序下载/启动→外设连接检查。
- 🔵 GPIO输入要有上拉或下拉默认电平，不能长期悬空。
- ⚫ 只改代码不能解决供电、复位或时钟硬件故障。

### 回忆卡21：中断、定时器与看门狗

先答：三者分别解决什么问题？

- 🔴 中断响应事件，定时器计时/周期触发/PWM，看门狗在程序失控时促使系统复位。
- 🔵 中断服务程序应短小，把耗时任务留给主循环；看门狗要按正常流程定期喂狗。
- ⚫ 中断不是越多越好，看门狗只能帮助恢复，不能证明软件没有缺陷。

### 回忆卡22：四种串行接口

先答：UART、I²C、SPI、CAN最小区别是什么？

- 🔴 UART异步点对点；I²C两线带地址；SPI同步高速且有片选；CAN多节点差分并有仲裁与错误检测。
- 🔵 选择时比较距离、速率、节点数、线数、同步方式和抗干扰。
- ⚫ I²C通常需上拉，CAN总线端需匹配；没有一种接口在所有场景都最好。

### 回忆卡23：按键消抖与PWM

先答：按键为什么要消抖，PWM占空比表示什么？

- 🔴 机械触点会在一次按下时产生多次跳变；PWM占空比是高电平时间占周期的比例。
- 🔵 消抖可用硬件滤波或软件状态确认；PWM常用于调光和调速。
- ⚫ PWM改变平均作用量，不等于输出理想连续直流；消抖不能粗暴忽略所有快速事件。

### 回忆卡24：示波器与逻辑分析仪

先答：排查数字接口时两种仪器怎样分工？

- 🔴 示波器看真实电压、边沿和噪声；逻辑分析仪看多路0/1时序和协议数据。
- 🔵 先检查电源地与电平，再查时钟/波特率、接线、上拉/终端和数据格式。
- ⚫ 逻辑分析仪只按阈值判高低，不能替代示波器判断模拟幅值和信号完整性。
