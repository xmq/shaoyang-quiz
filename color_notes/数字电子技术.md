# 📕 数字电子技术 · 三色笔记

使用说明：🔴核心逻辑规则与必背结论；🔵方程、化简、设计、波形与状态题步骤；⚫器件条件和易错点。完整电路图与例题见“讲义”模块。

## 一、数制、编码与逻辑代数

### 1. 数制与常用编码

🔴【位权展开】：二进制转十进制按 `Σ(数码×2^位权)`；十进制整数转二进制用除2取余逆序。

🔵【分组转换】：二进制↔八进制每3位一组；二进制↔十六进制每4位一组，不足在高位补0。

🔴【BCD码】：8421 BCD每4位表示一个十进制数字；例如十进制59写成 `0101 1001`，不是59的纯二进制。

🔴【格雷码】：相邻码仅一位变化，常用于减少转换冒险；二进制转格雷为最高位不变、其余相邻二进制位异或。

### 2. 基本逻辑与公式

🔴【基本门】：与 `AB`，或 `A+B`，非 `Ā`；异或输入不同时为1，同或输入相同时为1。

🔴【常用律】：`A+A=A`，`AA=A`，`A+0=A`，`A·1=A`，`A+Ā=1`，`AĀ=0`。

🔴【吸收律】：`A+AB=A`，`A(A+B)=A`。

🔴【德摩根】：`overline(A+B)=ĀB̄`，`overline(AB)=Ā+B̄`。

🔵【代数化简】：展开/提取公因式 → 用吸收律 → 配反变量 → 检查是否还能减少项或变量。

## 二、逻辑函数表示与卡诺图

### 3. 最小项、最大项和标准式

🔴【最小项】：每个变量出现一次，取原变量或反变量；某输入组合下仅对应一个最小项为1。

🔴【标准与或式】：函数为1的最小项之和 `F=Σm(...)`；标准或与式用函数为0的最大项表示 `F=ΠM(...)`。

⚫【编号】：变量顺序必须按题目约定，通常高位在左；不要把卡诺图格序当自然二进制顺序。

### 4. 卡诺图化简

🔴【相邻性】：行列采用格雷码；左右边界、上下边界也相邻，四角可组成一圈。

🔴【圈1规则】：与或式圈1；圈大小必须为 `1、2、4、8...`；圈尽量大、数量尽量少，所有1必须覆盖。

🔵【写项】：圈内保持不变的变量留下，变化变量消去；变量恒1写原变量，恒0写反变量。

⚫【重叠与无关项】：圈可以重叠；约束中的无关项X可按0或1使用，但只有能简化时才利用。

## 三、门电路与组合逻辑

### 5. TTL/CMOS基本常识

🔴【CMOS】：静态功耗低、输入电阻高、噪声容限较好；输入端不能悬空。

🔴【三态门】：除0、1外还有高阻态Z，可用于总线共享；高阻态不是逻辑0。

⚫【TTL悬空】：传统TTL输入悬空常等效高电平，但工程上仍应明确连接；题目指定系列时按器件手册。

### 6. 组合电路分析与设计

🔵【分析步骤】：由电路逐级写表达式 → 化简 → 列真值表 → 说明逻辑功能。

🔵【设计步骤】：读题定义输入输出 → 列真值表 → 写并化简函数 → 选门/译码器/数据选择器实现 → 检查无效输入。

🔴【半加器】：`S=A⊕B，C=AB`。【全加器】：`S=A⊕B⊕Cin`，进位为任意两个输入同时为1。

🔴【译码器】：n位输入最多产生 `2^n` 路互斥输出；关注使能端和输出高/低有效小圆圈。

🔴【数据选择器】：`2^n`选1需要n位选择端；可把变量接选择端、剩余变量接数据端实现逻辑函数。

🔴【编码器】：把有效输入位置编码；优先编码器允许多个输入同时有效并按优先级输出。

### 7. 竞争与冒险

🔴【原因】：同一信号经不同路径传播延迟不同，输入变化时可能产生瞬态毛刺。

🔵【常用消除】：增加冗余项、输出端滤波、用选通脉冲避开毛刺；同步系统还应满足时序约束。

## 四、触发器

### 8. RS、JK、D、T特性

🔴【RS】：置位、复位、保持；禁用组合取决于门型。或非型高有效禁用 `S=R=1`，与非型低有效禁用 `S̄=R̄=0`。

🔴【JK】：`Q+=JQ̄+K̄Q`；`J=K=1` 翻转，无RS禁用状态。

🔴【D】：`Q+=D`，有效沿把D送入Q。

🔴【T】：`Q+=T⊕Q`；`T=0` 保持，`T=1` 翻转。

⚫【异步端】：异步置位/清零端有效时不等待时钟，通常优先级最高；有效电平看小圆圈或题目说明。

### 9. 波形题

🔵【五步法】：确认触发器类型 → 看上升沿/下降沿/电平触发 → 标出异步端有效区 → 从给定初态开始 → 只在有效时刻代特性方程更新。

⚫【陷阱】：边沿触发器不是时钟高电平期间连续透明；必须区分输入建立/保持时间与传播延迟。

## 五、时序逻辑与计数器

### 10. 时序电路分析

🔵【标准步骤】：写各触发器激励方程 → 代入特性方程得状态方程 → 写输出方程 → 列状态转换表 → 画状态图/时序图 → 说明功能。

🔴【Moore型】：输出只与状态有关。【Mealy型】：输出与状态和当前输入都有关。

🔴【自启动】：从任何无效状态都能在有限拍进入有效状态循环；设计后必须检查未用状态。

### 11. 寄存器与计数器

🔴【寄存器】：存储多位数据；移位寄存器可实现串/并转换、延时和序列产生。

🔴【计数容量】：n个触发器最多有 `2^n` 个状态；模M计数器最少n满足 `2^n≥M`。

🔴【同步计数器】：各级共用时钟，速度较高；异步计数器逐级触发，传播延迟累积。

🔵【任意进制设计】：确定状态数 → 选触发器数 → 编码与状态图 → 求激励方程 → 化简实现 → 检查自启动。

## 六、脉冲、存储器与数模转换

### 12. 555定时器

🔴【核心阈值】：典型内部比较参考为 `1/3VCC` 与 `2/3VCC`。

🔴【单稳态】：一个稳定状态，触发后输出一个定宽脉冲。【多谐振荡器】：无稳定状态，连续产生矩形波。【施密特】：双阈值整形与抗噪。

### 13. 半导体存储器

🔴【ROM】：掉电不丢失，主要读出固定/少改内容。【RAM】：可读写，掉电通常丢失。

🔴【容量扩展】：字扩展增加地址数量；位扩展增加每字位数；芯片数=`字数倍数×位数倍数`。

### 14. ADC与DAC

🔴【ADC过程】：采样保持 → 量化 → 编码；n位ADC有 `2^n` 个量化级，分辨率约为满量程/`2^n`。

🔴【奈奎斯特抽样】：带限信号最高频率fm，理想条件 `fs≥2fm`。

🔴【DAC】：把数字量转换为模拟量；常考倒T形电阻网络、分辨率、满量程和转换时间。

⚫【量化误差】：理想均匀量化误差绝对值通常不超过半个量化间隔。
