# 数字电子技术 · 三色回忆卡

> 先答后核对。🔴是逻辑规则、特性方程和必写结论；🔵是化简、设计、波形和状态题步骤；⚫是有效电平、器件条件与易错边界。完整电路图和例题见“讲义”。

## 一、数制、编码与逻辑代数

### 回忆卡 1｜数制与常用编码

先答：二进制怎样与十进制、八进制、十六进制互换？BCD码和格雷码的关键区别是什么？

- 🔴 二进制按`Σ(数码×2^位权)`转十进制；二进制与八/十六进制分别每3/4位分组。8421 BCD每4位表示一个十进制数位；格雷码相邻码只变一位。
- 🔵 十进制整数转二进制用除2取余逆序；二进制转格雷时最高位不变，其余位为相邻二进制位异或。
- ⚫ 十进制59的BCD为`0101 1001`，不是59的纯二进制；分组不足只在最高侧补0。

### 回忆卡 2｜基本逻辑与公式化简

先答：与、或、非、异或怎样定义？吸收律和德摩根律是什么？

- 🔴 与`AB`、或`A+B`、非`Ā`；异或输入不同为1。吸收律`A+AB=A，A(A+B)=A`；德摩根`overline(A+B)=ĀB̄，overline(AB)=Ā+B̄`。
- 🔵 化简按展开/提公因式 → 吸收 → 配反变量 → 再检查项数和变量数；必要时用真值表验算。
- ⚫ 同一表达式中的“+、·、横线”是逻辑运算，不是普通算术；反演时运算符和每个变量都要变换。

## 二、逻辑函数表示与卡诺图

### 回忆卡 3｜最小项、最大项与标准式

先答：最小项、最大项怎样编号？标准与或式、或与式分别由哪些项组成？

- 🔴 每个最小/最大项都含全部变量一次；函数为1的编号写`F=Σm(...)`，函数为0的编号写`F=ΠM(...)`。
- 🔵 按题目变量顺序把输入当二进制编号，列真值表后分别收集输出1或0的行。
- ⚫ 变量顺序决定编号；卡诺图格按格雷码排列，不能当自然二进制顺序读取。

### 回忆卡 4｜卡诺图化简

先答：卡诺图怎样圈组并从圈写出最简项？无关项何时使用？

- 🔴 行列按格雷码；圈大小为`1、2、4、8...`，可跨边界和四角相邻；圈内不变变量保留，变化变量消去。
- 🔵 与或式圈1 → 先圈必需和最大圈 → 覆盖全部1 → 按恒1原变量、恒0反变量写项 → 合并。
- ⚫ 圈可重叠；无关项X可当0或1，但只在能扩大圈并简化时使用，不能漏掉规定的1。

## 三、门电路与组合逻辑

### 回忆卡 5｜TTL、CMOS与三态门

先答：CMOS输入为何不能悬空？三态门的高阻态表示什么？

- 🔴 CMOS静态功耗低、输入电阻高，输入必须接确定电平；三态门除0、1外还有高阻态Z，可用于总线共享。
- 🔵 器件题先看系列、有效电平和输出结构，再判输入端处理及总线是否允许多源驱动。
- ⚫ 高阻态不是逻辑0；传统TTL悬空常近似高电平但工程上仍应明确连接，题目指定器件时按手册。

### 回忆卡 6｜组合电路分析与设计

先答：组合电路分析、设计分别按什么顺序？加法器、译码器、选择器的核心功能是什么？

- 🔴 半加器`S=A⊕B，C=AB`；全加器`S=A⊕B⊕Cin`并产生进位。n位译码器最多`2^n`路输出，`2^n`选1数据选择器需n位选择端。
- 🔵 分析：逐级写式 → 化简 → 真值表 → 功能；设计：定义输入输出 → 真值表 → 化简函数 → 选器件实现 → 检查无效输入。
- ⚫ 必须检查使能端和输出高/低有效；优先编码器允许多输入有效，普通编码器通常不允许。

### 回忆卡 7｜竞争与冒险

先答：组合电路为什么产生瞬态毛刺？常用消除方法有哪些？

- 🔴 同一输入经不同路径传播延迟不同，输入变化时各路径到达不同步，可能产生竞争冒险。
- 🔵 用卡诺图加冗余项、输出滤波或选通脉冲避开毛刺；同步系统在时钟采样前满足时序约束。
- ⚫ 逻辑表达式稳态正确不代表动态无冒险；不能把滤波当作所有高速电路的通用补救。

## 四、触发器

### 回忆卡 8｜RS、JK、D、T特性

先答：四类触发器的下一状态方程和特殊输入是什么？

- 🔴 或非RS的`S=R=1`禁用、与非RS低有效端同时为0禁用；JK有`Q+=JQ̄+K̄Q`，D有`Q+=D`，T有`Q+=T⊕Q`。
- 🔵 先确认触发器类型与端口有效电平，再把当前输入和现态代特性方程求次态。
- ⚫ 异步置位/清零有效时通常优先且不等待时钟；小圆圈表示低有效，不能忽略。

### 回忆卡 9｜触发器波形题

先答：给定时钟、输入和初态，怎样逐拍画出Q波形？

- 🔴 边沿触发器只在规定有效沿更新，电平触发器在有效电平区响应；异步端有效时立即控制输出。
- 🔵 判类型 → 标有效沿/电平 → 标异步区 → 写初态 → 在每个有效时刻代方程 → 考虑传播延迟画Q。
- ⚫ 边沿触发器不是时钟高电平期间连续透明；建立时间、保持时间和传播延迟含义不同。

## 五、时序逻辑与计数器

### 回忆卡 10｜时序电路分析

先答：怎样从电路得到状态表、状态图和功能？Moore与Mealy输出依赖什么？

- 🔴 Moore输出只依赖状态，Mealy输出依赖状态和当前输入；自启动要求任一无效状态有限拍进入有效循环。
- 🔵 写激励方程 → 代特性方程得状态方程 → 写输出方程 → 列状态转换表 → 画状态图/时序图 → 说明功能。
- ⚫ 设计或分析后必须检查未用状态；只列有效循环不能证明自启动。

### 回忆卡 11｜寄存器与计数器

先答：n个触发器最多有多少状态？模M计数器怎样确定触发器数并设计？

- 🔴 n个触发器最多`2^n`状态，模M最少n满足`2^n≥M`；同步计数器共用时钟，异步计数器传播延迟逐级累积。
- 🔵 定状态数 → 选触发器数 → 编码/状态图 → 求激励方程 → 化简实现 → 检查无效状态和自启动。
- ⚫ 移位寄存器还能做串并转换、延时和序列产生；状态容量够不等于逻辑已构成正确模值。

## 六、脉冲、存储器与数模转换

### 回忆卡 12｜555定时器

先答：555的两个核心阈值是什么？单稳、多谐和施密特分别输出什么？

- 🔴 典型比较阈值为`1/3VCC`与`2/3VCC`；单稳触发后输出定宽脉冲，多谐无稳定状态连续振荡，施密特用双阈值整形。
- 🔵 先判电容充放电路径和比较器翻转点，再分段求脉宽、周期或阈值。
- ⚫ 具体公式受外接结构和控制端影响；不能只背阈值而忽略电容初态及放电管状态。

### 回忆卡 13｜半导体存储器

先答：ROM、RAM的基本区别是什么？存储器怎样做字扩展和位扩展？

- 🔴 ROM掉电不丢失且主要读，RAM可读写但掉电通常丢失；字扩展增加地址数，位扩展增加每字位数。
- 🔵 芯片数=`目标字数/单片字数 × 目标位数/单片位数`；字扩展增加片选译码，位扩展并联地址和控制端。
- ⚫ 容量单位和地址线数要统一；片选逻辑必须保证同一地址只使能目标芯片组。

### 回忆卡 14｜ADC与DAC

先答：ADC的过程、量化级数、抽样条件是什么？DAC完成什么转换？

- 🔴 ADC为采样保持 → 量化 → 编码；n位有`2^n`级，分辨率约满量程/`2^n`；带限最高频率`fm`时理想`fs≥2fm`。DAC把数字量转为模拟量。
- 🔵 容量题先定满量程和位数，再求量化间隔、数字码或输出；DAC按权电阻/倒T网络逐位叠加。
- ⚫ 理想均匀量化误差绝对值通常不超过半量化间隔；抽样等号是理想下限，工程系统要留滤波裕量。
