# 模拟电子技术 · 三色回忆卡

> 先答后核对。🔴是器件状态、基本公式和必写结论；🔵是判区、计算和作答顺序；⚫是模型条件、饱和边界与常见误判。完整电路图和推导见“讲义”。

## 一、二极管与整流电路

### 回忆卡 1｜PN结与二极管模型

先答：二极管为何单向导电？常用模型怎样判通断？

- 🔴 正偏使势垒降低而导通，反偏使势垒升高而近似截止；理想模型导通压降0，恒压降模型硅管约0.7 V、锗管约0.2～0.3 V。
- 🔵 假设导通/截止 → 按模型求阳极、阴极电位和电流 → 检查结果是否支持假设。
- ⚫ 温度升高时正向压降通常下降、反向饱和电流增大；普通管反向击穿可能损坏，稳压管须限流工作。

### 回忆卡 2｜整流、滤波与故障

先答：半波、桥式整流平均值是什么？电容滤波怎样减小纹波？

- 🔴 忽略管压降时半波`Uo≈0.45U2`，桥式全波`Uo≈0.9U2`；桥式每半周两管导通。
- 🔵 先判每半周导通路径，再画输出；滤波电容在峰值充电、其余时间向负载放电，增大C或负载电阻通常减小纹波。
- ⚫ 选管要校验平均电流和最高反向电压；桥中一管开路常退化为半波，电容开路会使纹波显著增大。

## 二、三极管与基本放大器

### 回忆卡 3｜BJT三个工作区

先答：截止、放大、饱和区的结偏置和电流关系分别是什么？

- 🔴 截止区发射结未正偏且`IB≈IC≈0`；放大区发射结正偏、集电结反偏且`IC≈βIB`；饱和区两结正偏且`VCE≈VCE(sat)`。
- 🔵 先按放大区算`IC=βIB` → 由外电路求可提供的集电流和`VCE` → 电流超限或`VCE`过小则改判饱和；无正向基极驱动则截止。
- ⚫ 饱和后不能继续用`IC=βIB`求实际集电流；NPN/PNP的电压极性必须同时反向考虑。

### 回忆卡 4｜静态工作点与失真

先答：静态工作点为何要设置？截止失真和饱和失真分别怎样产生？

- 🔴 静态量为`IBQ、ICQ、VCEQ`；工作点过低或摆幅过大进入截止，过高或摆幅过大进入饱和。
- 🔵 先求直流通路与Q点，再叠加交流摆幅并检查输出两端是否触及截止/饱和边界。
- ⚫ 发射极电阻的直流负反馈稳定温度；旁路电容交流短路可提高增益，但不取消其直流稳定作用。

### 回忆卡 5｜共射、共集与共基

先答：三种基本组态的相位、增益和输入输出电阻特点是什么？

- 🔴 共射电压增益大且反相；共集电压增益约1、同相、输入高输出低；共基电流增益略小于1、输入低且高频性能好。
- 🔵 共射发射极交流接地时可用`Au≈−β(RC∥RL)/rbe`；先求交流等效负载，再算增益与相位。
- ⚫ 接入负载后有效负载为`RC∥RL`，增益绝对值通常下降；直流通路与交流通路不能混画。

### 回忆卡 6｜场效应管

先答：FET与BJT的控制方式有何区别？MOS管判区必须先确认什么？

- 🔴 FET是电压控制器件，栅极电流很小、输入电阻高；BJT通常视为电流控制器件。
- 🔵 先判N/P沟道和增强/耗尽型，再按`VGS、VDS`极性与阈值判断截止、可变电阻或恒流区。
- ⚫ 不能只凭某个端电压正负判区；题目给出的源极、漏极定义和器件类型必须同时检查。

## 三、频率响应与多级放大

### 回忆卡 7｜频率响应

先答：上下限频率和−3 dB点怎样定义？低频、高频增益为何下降？

- 🔴 通频带为`fL`至`fH`；截止点电压增益降至中频值的`1/√2`，即−3 dB。
- 🔵 电压增益分贝为`20log10|Au|`，功率增益为`10log10(Po/Pi)`；先求比值再取对数。
- ⚫ 低频下降主要因耦合/旁路电容容抗增大，高频下降主要因结电容、杂散电容和米勒效应。

### 回忆卡 8｜多级放大

先答：多级放大器的总增益怎样合成？为什么不能直接乘各级空载增益？

- 🔴 线性电压增益逐级相乘，分贝增益逐级相加。
- 🔵 从最后一级向前计算各级实际负载和增益，再完成相乘或dB相加。
- ⚫ 后级输入电阻构成前级负载；级间耦合还可能改变低频截止点和静态偏置。

## 四、集成运算放大器

### 回忆卡 9｜理想运放与虚短虚断

先答：理想运放有哪些参数假设？什么时候才能使用虚短、虚断？

- 🔴 理想运放开环增益、输入电阻和带宽无穷大，输出电阻为0；在线性负反馈下`u+≈u−，i+≈i−≈0`。
- 🔵 先沿反馈路径确认负反馈，再检查输出是否在电源允许范围内，满足后才列虚短、虚断方程。
- ⚫ 开环比较器、正反馈或输出饱和时不能套虚短；虚短不是两输入端真实短接。

### 回忆卡 10｜比例、加法、积分与微分

先答：反相、同相、加法、积分、微分电路的基本关系是什么？

- 🔴 反相`uo=−(Rf/Rin)ui`，同相`uo=(1+Rf/R1)ui`；反相加法`uo=−RfΣ(ui/Ri)`；积分`uo=−(1/RC)∫ui dt`，微分`uo=−RC·dui/dt`。
- 🔵 确认负反馈 → 写虚短虚断 → 对反相节点列KCL → 解输出 → 检查幅值与极性。
- ⚫ 公式基于理想线性区及相应初始条件；输出超过电源摆幅时实际进入饱和。

### 回忆卡 11｜比较器与滞回比较器

先答：单限比较器与滞回比较器怎样决定输出？后者为何抗噪更强？

- 🔴 单限比较器比较`u+`和`u−`后输出正/负饱和；滞回比较器用正反馈形成上、下两个阈值。
- 🔵 分别求输入上升和下降时的翻转阈值，再按初态沿输入变化方向画输出。
- ⚫ 滞回的两个翻转点不同；比较器处于非线性状态，不使用虚短。

## 五、反馈与振荡

### 回忆卡 12｜反馈类型与负反馈作用

先答：怎样判反馈极性和四种组态？负反馈怎样改变增益、带宽和电阻？

- 🔴 `Af=A/(1+AF)`；深度负反馈`|AF|≫1`时`Af≈1/F`。负反馈稳定增益、减小失真、扩展带宽但降低增益。
- 🔵 瞬时极性判正负 → 输出端判电压/电流取样 → 输入端判串联/并联叠加；串联提高输入电阻、并联降低，电压反馈降低输出电阻、电流反馈提高。
- ⚫ 只有环路工作在线性负反馈区才能套闭环公式；“改善电阻”方向取决于具体组态。

### 回忆卡 13｜正弦振荡

先答：正弦自激振荡的幅值、相位条件和基本组成是什么？

- 🔴 稳态满足`|AF|=1`且总相移`2kπ`；电路含放大、正反馈、选频和稳幅环节。
- 🔵 先检查相位条件，再算环路增益；起振时通常使`|AF|>1`，振幅增长后由稳幅环节回到1。
- ⚫ RC适合低频、LC适合高频、晶体稳定度高；满足相位条件但环路增益不足仍不能起振。

## 六、功率放大与直流稳压

### 回忆卡 14｜功率放大

先答：甲类、乙类、甲乙类的效率与失真如何比较？互补对称电路怎样工作？

- 🔴 甲类失真小效率低；乙类静态功耗小效率高但有交越失真；甲乙类用小静态偏置减小交越失真。互补管分别放大正、负半周。
- 🔵 功放题依次求最大输出摆幅、输出功率、直流功率、效率和管耗，并校验安全工作区。
- ⚫ 最大管耗不一定出现在最大输出功率点；不能只按负载功率判断晶体管安全。

### 回忆卡 15｜整流滤波与稳压链路

先答：直流稳压电源各级作用是什么？稳压管和串联反馈稳压怎样稳定输出？

- 🔴 功能链为变压 → 整流 → 滤波 → 稳压；稳压管反向击穿，串联电阻限流分压；串联稳压用取样比较、误差放大和调整管构成负反馈。
- 🔵 先按负载电流和输入波动求串联电阻或调整管工作范围，再校验最低、最高输入下的输出和器件功耗。
- ⚫ 稳压管电流必须处于最小稳压电流与最大允许电流之间；输入过低或负载过重会失去稳压。
