# 模拟电子技术｜三色回忆卡

先答后核对：红色写结论，蓝色写原因或判断依据，黑色写成立边界与易错点。

## 一、半导体与二极管

### 回忆卡1：模拟信号与放大

先答：什么是模拟信号，放大器的输出能量来自哪里？


- ⚫ 模拟信号幅度连续变化；放大器输出能量来自直流电源。
- 🔵 简答写“微小输入控制较大输出”，再说明输入只起控制作用。
- 🔴 放大不是凭空制造能量，也不等于输出一定没有失真。
### 回忆卡2：P型、N型与PN结

先答：P型、N型多数载流子是什么，PN结有什么基本特性？


- ⚫ P型多数载流子为空穴，N型为电子；PN结具有单向导电性。
- 🔵 用“正偏势垒降低易导通、反偏势垒升高通常截止”解释。
- 🔴 P型和N型整体都保持电中性，不能说分别带正电和负电。
### 回忆卡3：常见二极管

先答：整流、稳压、发光和光电二极管分别做什么？


- 🔴 依次用于整流、稳压、发光、光电转换。
- 🔵 简答采用“名称＋主要功能”逐项列举。
- ⚫ 稳压管利用反向击穿但必须限流；LED正向发光也需要限流。
### 回忆卡4：直流稳压电源

先答：直流稳压电源通常有哪些基本环节？


- ⚫ 变压、整流、滤波、稳压。
- 🔵 对应作用：变换电压、交流转脉动直流、减小脉动、稳定输出。
- 🔴 滤波只能减小纹波，不等于已经完成稳压。
## 二、三极管与放大器

### 回忆卡5：三极管三种状态

先答：三极管截止、放大、饱和状态各有什么特点？


- 🔴 截止近似断开，放大区用于线性放大，饱和近似闭合。
- 🔵 放大区可写“发射结正偏、集电结反偏”。
- ⚫ `IC=βIB` 主要用于放大区，不能机械套到饱和区。
### 回忆卡6：静态工作点

先答：放大电路为什么要设置合适的静态工作点？


- 🔴 为三极管提供合适直流偏置，使信号在放大区内摆动。
- 🔵 补充：工作点过低易截止失真，过高易饱和失真。
- ⚫ 静态工作点描述无交流输入时的直流状态，不是信号最大值。
### 回忆卡7：三种基本组态

先答：共射、共集、共基放大器各有什么突出特点？


- 🔴 共射放大能力强且反相；共集增益约1、适合缓冲；共基高频性能较好。
- 🔵 对比按“增益—相位—输入输出电阻—用途”组织。
- ⚫ “共”指输入输出回路共用电极，不一定直接接物理地。
### 回忆卡8：放大器基本指标

先答：描述放大器常用哪些基本指标？


- 🔵 增益、输入电阻、输出电阻、通频带、失真。
- 🔴 简答可解释：输入电阻反映对信号源影响，输出电阻反映带负载能力。
- ⚫ 电压放大器通常希望输入电阻大、输出电阻小，但不是所有放大器指标都相同。
### 回忆卡9：MOSFET与BJT

先答：MOSFET有什么控制特点，增强型MOSFET怎样判断截止与导通，它与BJT有何区别？


- ⚫ MOSFET主要由栅极电压控制、输入电阻高；增强型在栅源电压不足时截止，超过阈值后可导通。
- 🔵 对比记忆：MOSFET是电压控制，BJT主要由基极电流控制；两者都能放大和开关。
- 🔴 超过阈值只表示开始形成沟道，不等于已经成为无压降的理想开关。
## 三、反馈与运算放大器

### 回忆卡10：正反馈与负反馈

先答：怎样区分正反馈和负反馈？


- ⚫ 反馈使净输入减小是负反馈，使净输入增大是正反馈。
- 🔵 判断时比较引入反馈前后的净输入，而不是只看接线端名称。
- 🔴 正反馈并非一律错误，它常用于振荡或产生迟滞。
### 回忆卡11：负反馈的影响

先答：负反馈通常对放大器产生哪些影响？


- ⚫ 降低并稳定增益、减小失真、展宽通频带，并改变输入输出电阻。
- 🔵 简答先写“牺牲部分增益换稳定性”，再列三到四项影响。
- 🔴 负反馈不能消除所有噪声，也不是必然提高放大倍数。
### 回忆卡12：运放的虚短和虚断

先答：线性运放中“虚短”“虚断”分别是什么意思？


- 🔵 虚短：两输入端电位近似相等；虚断：两输入端电流近似为0。
- 🔴 使用前先确认深度负反馈且输出没有饱和。
- ⚫ 虚短不是真短路，虚断也不表示器件内部完全没有电流。
### 回忆卡13：比较器和电压跟随器

先答：比较器与电压跟随器各有什么功能？


- ⚫ 比较器判断两输入大小并输出高低电平；跟随器输出约等于输入，起缓冲作用。
- 🔵 对比关键是“比较器多为非线性，跟随器为深度负反馈线性电路”。
- 🔴 比较器输出通常接近饱和值，不能套虚短直接作线性计算。
## 四、常见判断与简答

### 回忆卡14：二极管单向导电简答

先答：简述二极管单向导电性的原因。


- 🔴 PN结正偏易导通、反偏通常截止，因此表现为单向导电。
- 🔵 展开写势垒在正偏时降低、反偏时升高。
- ⚫ 反向电压超过限值可能击穿，不能说反向永远绝对无电流。
### 回忆卡15：三极管作开关

先答：三极管作电子开关时主要使用哪两个状态？


- 🔴 截止和饱和。
- 🔵 截止对应开关断开，饱和对应开关闭合。
- ⚫ 放大区用于线性放大，不是理想开关的两个稳定状态之一。
### 回忆卡16：反相与同相运放直接计算

先答：两种基本运放的闭环增益公式是什么，什么时候不能继续套用？


- ⚫ 反相 `Au=-Rf/R1`；同相 `Au=1+Rf/R1`。
- 🔵 先辨认拓扑，再算增益与输出；负号表示反相，不表示幅值为负。
- 🔴 公式要求负反馈且运放在线性区；计算输出超过允许摆幅时，实际会饱和。
### 回忆卡17：为什么电压跟随器能作缓冲

先答：电压跟随器增益约为1，为什么仍然有用？


- ⚫ 它输入电阻高、输出电阻低，能隔离前后级并增强带负载能力。
- 🔵 原因链写“少取前级电流—输出电压跟随—向后级供给较大电流”。
- 🔴 它不提高电压幅值，且输出电流、摆幅和频率仍受运放规格限制。
## 五、检测、波形与采集

### 回忆卡18：二极管和三极管怎样初检

先答：万用表二极管挡能判断什么，不能证明什么？


- 🔴 正常PN结通常正向有压降、反向截止；BJT可按两个PN结初步检查。
- 🔵 两向都近0疑似短路，两向都截止疑似开路；在路结果异常时脱开一端复核。
- ⚫ 初检不能证明放大倍数、耐压和高频性能都合格，且必须断电测量。
### 回忆卡19：MOSFET为什么怕静电

先答：MOSFET拿取和焊接为什么要防静电？


- ⚫ 栅极绝缘层薄，静电高压可能击穿，应使用接地腕带和防静电工作面。
- 🔵 运输时保持引脚等电位，操作时避免直接触摸栅极并规范接地。
- 🔴 防静电不等于带电操作；接地设备也必须符合人身安全要求。
### 回忆卡20：看削顶判断什么

先答：放大器输出双侧削顶和单侧削顶各提示什么？


- 🔵 双侧削顶多提示输入过大或摆幅不足；单侧削顶多提示静态工作点偏向截止或饱和。
- 🔴 先测供电和静态工作点，再逐级比较输入、输出波形。
- ⚫ 万用表平均值正常不代表波形无失真；示波器无交流也不等于直流为0。
### 回忆卡21：传感器采集链

先答：传感器信号进入ADC前为什么要放大和滤波？


- 🔵 链路为传感器→保护/偏置→放大→抗混叠滤波→ADC→处理。
- 🔴 放大提高量程利用率，滤波限制带宽并减少混叠，缓冲可减小对高源阻传感器的负载。
- ⚫ 放大过大可能饱和；提高ADC位数或采样率不能自动消除前端噪声。
### 回忆卡22：感性负载的续流二极管

先答：直流继电器线圈为什么常并联续流二极管？

- 🔴 控制管关断时线圈产生反向感应电压，二极管提供续流通路并保护开关管。
- 🔵 二极管正常通电时反向截止，断电反冲时导通释放磁场能量。
- ⚫ 极性接反会在正常通电时近似短路；交流线圈不能机械照搬直流续流接法。
