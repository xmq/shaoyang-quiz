# 操作系统原理 · 三色回忆卡

> 使用方法：遮住答案，先口述“先答”，再核对。🔴只记最小结论；🔵只记计算或算法步骤；⚫只记成立条件与反例。

## 一、系统、进程与调度

### 回忆卡 1｜操作系统总览

先答：操作系统的四项功能、四个特征，以及系统调用与中断/异常的关系是什么？

- 🔴 功能是处理机、存储、文件、设备管理；特征是并发、共享、虚拟、异步。系统调用是用户程序主动请求内核服务的接口，中断/异常是进入内核的机制。
- 🔵 辨析顺序：来源在程序内/外 → 是否同步于当前指令 → 是否主动请求服务。
- ⚫ 并发不等于同一时刻并行；库函数不一定触发系统调用；缺页是异常，I/O 完成通常是中断。

### 回忆卡 2｜进程、线程与状态

先答：进程与线程分别是什么单位？三状态怎样转换？

- 🔴 进程是资源分配和保护单位，PCB 是其存在标志；线程是 CPU 调度单位，同进程线程共享地址空间和文件，但各有 PC、寄存器和栈。
- 🔵 就绪经调度到运行；运行因抢占回就绪、因等事件到阻塞；事件完成使阻塞到就绪。
- ⚫ 阻塞不能直接到运行；进程切换通常比同进程线程切换开销大。

### 回忆卡 3｜调度指标

先答：怎样由到达、服务、首次运行和完成时刻求四类调度指标？

- 🔴 周转＝完成－到达；带权周转＝周转/服务；响应＝首次运行－到达；无额外阻塞时等待＝周转－服务。
- 🔵 先画甘特图，再列首次与完成时刻，逐进程计算后求平均。
- ⚫ 抢占式算法须在到达、完成等决策点重选，不能一次排到底。

### 回忆卡 4｜调度算法

先答：FCFS、SJF/SRTF、优先级和 RR 各按什么规则选择，主要代价是什么？

- 🔴 FCFS 按到达；SJF 选最短服务、SRTF 选最短剩余；优先级选最高优先；RR 按时间片轮转。
- 🔵 手算统一做“维护就绪队列 → 决策点选择 → 甘特图 → 指标表”。
- ⚫ FCFS 有护航效应；SJF/优先级会饥饿；RR 时间片过大趋于 FCFS，过小切换开销大。

## 二、同步与死锁

### 回忆卡 5｜同步、互斥与信号量

先答：同步、互斥、临界区和 P/V 分别解决什么问题？

- 🔴 同步约束先后，互斥禁止同时访问临界资源；P 在资源不足时等待，V 释放资源或发出完成信号。
- 🔵 PV 题先找先后约束与资源容量，再设初值，最后把 P 放在等待/使用前、V 放在释放/通知后。
- ⚫ 记录型信号量可阻塞而非忙等；P/V 不配对或持锁等待另一资源可能死锁。

### 回忆卡 6｜生产者—消费者

先答：容量为 n 的缓冲区，三个信号量初值和生产、消费顺序是什么？

- 🔴 `empty=n，full=0，mutex=1`；生产者 P(empty)、P(mutex)、放入、V(mutex)、V(full)，消费者对称。
- 🔵 按“资源计数 → 互斥进入 → 操作 → 解锁 → 通知”检查两条路径。
- ⚫ 先 P(mutex) 再等待 empty/full，可能占锁等待造成死锁。

### 回忆卡 7｜读者—写者与管程

先答：读者—写者的最小约束是什么？管程和条件变量做什么？

- 🔴 读者可并发，写者与任何读写互斥；管程封装共享数据并保证内部过程互斥，条件变量负责等待/唤醒。
- 🔵 常用 readcount：首个读者锁写资源，末个读者释放；写优先还要阻止新读者插队。
- ⚫ 条件变量不是保存资源数的计数信号量；不同公平策略的信号量设计不同。

### 回忆卡 8｜死锁与银行家

先答：死锁四条件是什么？怎样判安全状态并处理新请求？

- 🔴 四条件是互斥、请求并保持、不剥夺、循环等待；安全状态存在安全序列，不安全不等于已死锁。
- 🔵 `Need=Max-Allocation`，`Work=Available`；反复找 `Need≤Work` 的进程，完成后 `Work+=Allocation`。请求先查不超 Need/Available，再试分配并重做安全检查。
- ⚫ 向量的每个分量都要满足；完成后回收 Allocation，不是 Need。

## 三、内存与虚拟存储

### 回忆卡 9｜连续分配与碎片

先答：首次、最佳、最坏适应怎样选分区？内部与外部碎片怎样区分？

- 🔴 首次取首个足够块，最佳取最小足够块，最坏取最大块；分区内部未用是内部碎片，空闲区离散是外部碎片。
- 🔵 分配题逐次更新空闲表，保留起址、长度和合并结果。
- ⚫ “剩余最小”不保证长期利用最好；分页仍可能有页内碎片。

### 回忆卡 10｜分页地址变换

先答：页大小为 2^k 时，怎样拆逻辑地址并求物理地址？

- 🔴 偏移占 k 位；页号 `p=地址÷页大小`，偏移 `d=地址 mod 页大小`，查页表得页框 f，物理地址 `f×页大小+d`。
- 🔵 先验页号合法和有效位，再查表、拼接；二进制题直接切“页号｜偏移”。
- ⚫ 页号越界与缺页不同；页内偏移不经转换。

### 回忆卡 11｜分段、段页式与越界

先答：分段地址怎样检查和变换？它与分页最小区别是什么？

- 🔴 段号查段表，若偏移小于段长则物理地址＝段基址＋偏移；分页定长且对用户透明，分段变长并反映逻辑。
- 🔵 段页式先用段号取得页表，再用页号取页框，最后拼偏移。
- ⚫ 偏移等于段长也越界；分段有外部碎片，分页主要有内部碎片。

### 回忆卡 12｜TLB 与有效访问时间

先答：TLB 命中、未命中和缺页分别访问哪里？有效访问时间怎样列？

- 🔴 TLB 命中直接得页框；未命中查内存页表；缺页才访问外存。
- 🔵 按题设画概率分支：每条路径把 TLB、页表、主存和缺页服务时间相加，再乘概率求期望。
- ⚫ 串行查 TLB 时，常见命中为 `tTLB+tM`、未命中且在内存为 `tTLB+2tM`；并行模型不能套此式。

### 回忆卡 13｜页面置换

先答：OPT、FIFO、LRU、CLOCK 淘汰谁？怎样手算缺页率？

- 🔴 OPT 淘汰未来最久不用，FIFO 淘汰最早装入，LRU 淘汰过去最久未用，CLOCK 循环找访问位 0。
- 🔵 每次访问一列、每个页框一行；逐列更新状态与辅助队列/访问位，初装也记缺页，最后缺页次数除访问次数。
- ⚫ FIFO 可能有 Belady 异常；页框不足导致频繁换页称抖动，可用工作集/缺页率控制。

## 四、文件与 I/O

### 回忆卡 14｜文件分配与索引容量

先答：连续、链接、索引分配各适合什么访问？多级索引最大文件怎样算？

- 🔴 连续分配快但扩展难；链接易扩展但随机访问差；索引支持随机访问但有索引块开销。
- 🔵 每索引块地址数＝块大小/块号长度；把直接、一级、二级、三级可寻址数据块数相加，再乘数据块大小。
- ⚫ inode 与索引块通常不计入文件数据长度；题设若规定混合索引数量，逐项代入。

### 回忆卡 15｜目录、链接与空闲空间

先答：硬链接、符号链接和位示图的关键区别是什么？

- 🔴 硬链接让多个目录项指向同一 inode；符号链接保存目标路径；位示图每位表示一个盘块状态。
- 🔵 位示图题先按题设确认 0/1 语义，再由盘块号求字/位位置或反算。
- ⚫ inode 不保存文件名；硬链接通常不能跨文件系统，符号链接目标删除后会悬空。

### 回忆卡 16｜I/O、DMA 与缓冲

先答：查询、中断、DMA 三种 I/O 控制中 CPU 分别参与到什么程度？

- 🔴 查询让 CPU 轮询；中断由设备完成后通知；DMA 控制器在设备与主存间成块传输，CPU 负责初始化和结束处理。
- 🔵 比较题按“CPU 等待、每次数据是否经 CPU、并行度、硬件代价”回答。
- ⚫ DMA 仍争用总线/主存，不是 CPU 完全不参与；缓冲改善速度匹配，不改变设备本身速度。

### 回忆卡 17｜磁盘调度

先答：FCFS、SSTF、SCAN/LOOK、C-SCAN/C-LOOK 怎样确定访问序列？

- 🔴 FCFS 按请求序；SSTF 取最近；SCAN 到端点再反向，LOOK 到本方向最远请求；循环版本只沿一个方向服务。
- 🔵 写初始磁头、方向和端点规则，列序列，再累加相邻磁道差绝对值。
- ⚫ SSTF 可能使远端饥饿；SCAN 是否到物理端点按题设，LOOK 不走无请求端点。

## 五、闭卷大题动作

### 回忆卡 18｜过程题最低得分结构

先答：调度、PV、银行家、置换和磁盘题至少保留哪些过程？

- 🔴 必须写初始状态、逐轮选择、状态更新、终止条件和最终结果。
- 🔵 分别保留甘特图、P/V 初值与位置、安全序列、页框表、磁道访问序列。
- ⚫ 只写最终平均值、缺页数或总移动量，通常会丢主要步骤分；教材规则有变体时先声明。
