# 多媒体技术：三色笔记

三色按考频和出题稳定性划分：红色优先背熟，蓝色用于补齐得分点，黑色用于防漏和边界判断。

## 🔴 高频考点

- 🔴 **媒体类型**：文本、图形、图像、音频、视频、动画是常见媒体。 常见考法：判断媒体类别和特点。 易错边界：图形是矢量，图像通常是位图。
- 🔴 **数字化过程**：采样、量化、编码把连续信号变成数字数据。 常见考法：排序过程，解释模拟转数字。 易错边界：采样解决时间离散，量化解决幅度离散。
- 🔴 **图像参数**：分辨率、颜色深度决定未压缩图像大小。 常见考法：计算图像容量。 易错边界：颜色深度单位是 bit，不是 Byte。
- 🔴 **音频参数**：采样率、量化位数、声道数决定 PCM 码率。 常见考法：计算音频文件大小。 易错边界：立体声要乘声道数。
- 🔴 **视频参数**：帧率、分辨率、码率影响清晰度和容量。 常见考法：按码率和时长求文件大小。 易错边界：码率相同不代表画质一定相同，还受编码效率影响。
- 🔴 **压缩**：无损压缩可还原， 有损压缩牺牲部分信息换更高压缩比。 常见考法：判断 ZIP、JPEG、MP3、H.264。 易错边界：有损压缩多次处理会累积损失。

## 🔵 中频考点

- 🔵 **色彩模型**：RGB 面向显示，CMYK 面向印刷，HSV 更贴近色相调节。 常见考法：判断应用场景。 易错边界：屏幕颜色和印刷颜色可能有差异。
- 🔵 **常见格式**：JPG、PNG、GIF、MP3、MP4 等格式用途不同。 常见考法：按透明、动画、压缩选择格式。 易错边界：文件扩展名不能完全证明真实编码。
- 🔵 **流媒体**：边下载边播放，依赖网络带宽和缓冲。 常见考法：判断卡顿原因。 易错边界：缓存不是永久下载保存。
- 🔵 **多媒体硬件**：声卡、显卡、摄像头、采集卡负责采集和处理。 常见考法：判断设备功能。 易错边界：麦克风采集的是声音信号，不是直接生成文本。
- 🔵 **交互媒体**：超文本、超媒体支持非线性浏览。 常见考法：概念判断。 易错边界：交互性是多媒体系统的重要特点。

## ⚫ 低频了解

- ⚫ **版权**：素材使用要考虑授权和版权。 常见考法：判断是否可随意商用。 易错边界：免费可下载不等于免费商用。
- ⚫ **VR/AR**：VR 强调虚拟环境，AR 强调现实叠加。 常见考法：概念区分。 易错边界：二者都依赖显示、跟踪和交互技术。
- ⚫ **多媒体数据库**：管理图像、音频、视频等非结构化数据。 常见考法：概念判断。 易错边界：检索常依赖元数据和特征。

## 考前公式与简答

### 必会公式

- **未压缩位图容量**：`容量(Byte)=宽度×高度×颜色深度(bit)/8`。宽度和高度表示像素数，颜色深度表示每个像素占多少位，除以8是把bit换成Byte；题目若有多帧还要乘帧数。
- **PCM音频容量**：`容量(Byte)=采样率×量化位数×声道数×时长/8`。采样率单位Hz，量化位数单位bit，立体声通常为2声道；结果再按1024或1000换算KB、MB要看题目约定。
- **按码率估算媒体容量**：`容量(Byte)=码率(bit/s)×时长(s)/8`。码率已包含压缩后的每秒数据量时，不再重复乘分辨率、帧率或声道数。

### 常见简答题

- **采样与量化有什么区别？** 采样把连续时间离散化，决定时间方向的分辨能力；量化把连续幅值映射到有限等级，决定幅值分辨能力并产生量化误差。
- **无损压缩与有损压缩如何选择？** 文档、程序和需精确还原的数据使用无损压缩；图像、音频、视频在允许感知损失时可用有损压缩换取更高压缩比。
