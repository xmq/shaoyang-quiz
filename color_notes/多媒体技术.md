# 多媒体技术 · 三色回忆卡

> 先答后核对。容量题先统一bit/Byte和时间单位，KB/MB进制按题干约定。

## 一、多媒体概念与组成

### 回忆卡 1｜定义与特点

先答：什么是多媒体技术？主要特点有哪些？


- 🔴 它对文本、图形图像、音频、动画和视频进行获取、处理、集成、传输与展示；特点是多样、集成、交互、数字化和实时。
- 🔵 简答先定义，再用“特点＋解释”分点写。
- ⚫ 多种媒体同时出现不必然形成有意义的交互式多媒体系统。
### 回忆卡 2｜媒体类型

先答：文本、图形、图像、音频、动画和视频分别怎样表示信息？


- 🔴 文本用字符，图形用几何描述，图像用像素，音频用声音数据，动画用连续画面，视频用连续图像帧并常含声音。
- 🔵 分类题先说明对象，再写基本表示方式和用途。
- ⚫ 图形与图像不能混同；动画和视频也不是完全相同概念。
### 回忆卡 3｜超文本、超媒体与系统组成

先答：超文本与超媒体有何区别？多媒体系统由哪些层次组成？


- ⚫ 超文本链接文本节点，超媒体把链接对象扩展到多种媒体；系统包括硬件、系统软件、处理/创作软件、数据和应用。
- 🔵 区别题写“组织方式相同＋媒体范围不同”。
- 🔴 超媒体包含超文本思想，不代表所有内容都必须按固定顺序阅读。
## 二、数字化与容量

### 回忆卡 1｜采样、量化、编码

先答：声音从声波进入计算机要经过什么链路？采样、量化、编码各做什么？


- 🔵 声波 → 麦克风的模拟电信号 → 放大/抗混叠滤波 → ADC；其中采样离散时间，量化离散幅值，编码形成数字代码。
- 🔴 n位可表示`2^n`个状态；计算机以电平承载0/1，便于半导体开关完成运算和存储。
- ⚫ 数字数据不是“没有物理信号”，仍需电、光或无线电承载；提高采样率和位数也不能恢复源中已丢失的信息。
### 回忆卡 2｜数据单位

先答：bit、Byte、码率分别表示什么？


- 🔵 bit是二进制位，1 Byte通常等于8 bit；码率是单位时间的数据量，常用bit/s。
- 🔴 计算前统一bit与Byte、秒与分钟，并按题干选择1000或1024进制。
- ⚫ 位和字节相差8倍，不能只改单位名称而不换算数值。
### 回忆卡 3｜三类容量关系

先答：位图、PCM音频和给定码率的音视频分别怎样估算容量？


- 🔴 位图=`宽×高×位深÷8`；PCM=`采样率×位数×通道×时间÷8`；码率媒体=`总码率×时间÷8`。
- 🔵 例：8 kHz、8 bit、单声道、10秒PCM为`8000×8×1×10÷8=80 kB`；按2 Mbit/s保存60秒为`2×60÷8=15 MB`（均按1000进制）。
- ⚫ 已给总码率时通常不再乘像素、位深和帧率；实际文件还可能有文件头、索引等开销。
## 三、数字图像

### 回忆卡 1｜位图与矢量图

先答：两者的组成、缩放表现和适用对象有何不同？


- 🔴 位图由像素组成，适合照片且放大会失真；矢量图由几何描述，适合标志图形且便于缩放。
- 🔵 比较按“组成—缩放—用途—大小影响因素”写。
- ⚫ 矢量图不是由更小像素组成，位图也不是任何情况下都优于矢量图。
### 回忆卡 2｜颜色模型

先答：RGB、CMYK、HSV/HSB和YCbCr各用于什么？


- 🔴 RGB用于发光显示，CMYK用于印刷，HSV便于调色，YCbCr分离亮度色度并常用于图像视频。
- 🔵 选择模型先看输出介质和处理目的。
- ⚫ 屏幕RGB颜色不保证印刷后完全一致。
### 回忆卡 3｜常见格式

先答：JPEG、PNG、GIF、SVG、PSD的核心区别是什么？


- 🔴 JPEG有损且适合照片；PNG无损并支持透明；GIF色彩有限且可简单动画；SVG是矢量；PSD保留编辑结构。
- 🔵 选择格式看照片/图标、透明、动画、无损和是否继续编辑。
- ⚫ JPEG不适合反复无损保存，导出图像也不能代替可编辑源文件。
## 四、音频、动画与视频

### 回忆卡 1｜数字音频

先答：音频的主要参数是什么？WAV、MP3、FLAC、MIDI怎样区别？


- 🔵 参数有采样率、位数、通道和码率；WAV是容器且常装PCM，MP3常有损，FLAC无损，MIDI记录演奏事件。
- 🔴 格式题写“是否波形录音—是否压缩—能否恢复—典型用途”。
- ⚫ WAV不一定无压缩，MIDI也不是人的声音波形录音。
### 回忆卡 2｜动画

先答：逐帧、运动补间和形状补间分别适合什么？


- ⚫ 逐帧逐个制作画面；运动补间改变位置缩放等；形状补间用于矢量形状变化。
- 🔵 时长=`总帧数÷帧率`，总帧数=`时长×帧率`。
- 🔴 关键帧不等于每一帧都有新画面，帧率越高也会增加数据和处理量。
### 回忆卡 3｜视频编码与容器

先答：分辨率、帧率、码率分别表示什么？编码器与容器有何区别？


- 🔵 分辨率看像素量，帧率看每秒帧数，码率看每秒数据量；编码器规定压缩，容器封装音视频字幕等。
- 🔴 比较视频不能只看分辨率，还要看帧率、码率、编码和源质量。
- ⚫ MP4主要是容器，不唯一决定内部编码；同分辨率不等于同质量。
## 五、压缩、应用与基础答题

### 回忆卡 1｜无损与有损压缩

先答：两类压缩的关键区别和适用对象是什么？


- ⚫ 无损可恢复原数据，适合文本程序和编辑素材；有损牺牲部分信息换高压缩比，常用于照片、音视频分发。
- 🔵 压缩比常为`原始量÷压缩后量`，先确认题干定义。
- 🔴 有损压缩不是文件损坏，但反复有损保存会继续降低质量。
### 回忆卡 2｜作品流程与原则

先答：多媒体作品从需求到发布有哪些环节？


- ⚫ 需求受众 → 结构脚本 → 素材 → 编辑集成 → 测试修改 → 发布维护。
- 🔵 检查内容正确、重点突出、导航一致、文字可读、版权隐私和设备兼容。
- 🔴 动画装饰越多不代表效果越好，网上可下载也不代表可自由使用。
### 回忆卡 3｜简答与判断题

先答：定义特点题、区别题、格式选择题怎样组织答案？

- ⚫ 定义特点写“定义＋特点解释”；区别按组成/用途/优缺点；格式选择写需求和理由。
- 🔵 判断题找对象和必要条件，再用正确表述替换错误句。
- 🔴 绝对化词只是提醒，不能不分析知识点就机械判错。
