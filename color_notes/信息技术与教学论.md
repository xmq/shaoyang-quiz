# 信息技术与教学论 · 三色回忆卡

> 先答后核对，重点提取定义、特点、分类、区别、简答关键词和判断条件。

## 一、课程与学习理论

### 回忆卡 1｜信息技术教育

先答：信息技术教育的基本作用和主要特点是什么？


- 🔴 作用是发展信息意识、信息处理、计算思维、实践创新和责任；特点是基础性、实践性、综合性、发展性和创新性。
- 🔵 简答先定义，再用“特点＋一句解释”分点作答。
- ⚫ 信息技术教育不只是软件操作，实践性也不等于自由上机。
### 回忆卡 2｜基本素养

先答：信息意识、计算思维、数字化学习与创新、信息社会责任分别强调什么？


- 🔴 依次强调信息需求和判断、分解抽象算法、工具资源与创作、法律伦理隐私安全。
- 🔵 案例题先识别行为，再对应素养并写证据。
- ⚫ 计算思维不等于只会编程，使用设备不等于形成数字化学习能力。
### 回忆卡 3｜四种学习理论

先答：行为、认知、建构和人本主义各关注什么？


- ⚫ 依次关注练习强化、信息加工、情境协作中的意义建构、人的需要和自我实现。
- 🔵 匹配：操作训练—行为；知识结构—认知；真实任务—建构；情感选择—人本。
- 🔴 建构主义不要求教师退出，任何理论都不是所有内容的唯一方法。
## 二、教学设计

### 回忆卡 1｜设计要素与流程

先答：教学设计包括哪些基本要素和环节？


- 🔵 分析学习者与内容 → 定目标 → 设计活动、方法、资源和评价 → 实施 → 反馈修改。
- 🔴 简答踩分词：学情、内容、目标、活动、资源、环境、评价、改进。
- ⚫ 只写教学步骤而无目标和评价，不是完整教学设计。
### 回忆卡 2｜学情、重点与难点

先答：学情分析看什么？重点为什么不一定是难点？


- 🔵 学情看先备知识、认知特点、动机经验、差异和设备条件；重点由要求与内容价值决定，难点由内容和学生基础共同决定。
- 🔴 用前测、观察、访谈、作品等证据判断，不只凭教师印象。
- ⚫ 重点与难点依据不同，不能默认二者完全重合。
### 回忆卡 3｜目标与一致性

先答：怎样写可评价目标？“教—学—评一致”是什么意思？


- 🔵 目标写学习者、可观察行为、条件和标准；目标、学习活动与评价证据必须相互对应。
- 🔴 句式：“在……条件下，学生能……，达到……标准。”
- ⚫ 只写了解、掌握、培养而没有行为证据，通常无法直接评价。
## 三、教学方法与课堂组织

### 回忆卡 1｜讲授与演示练习

先答：讲授法和演示练习法各适合什么？


- ⚫ 讲授适合系统说明概念原理；演示练习适合操作技能，强调示范、模仿、独立练习和迁移。
- 🔵 选择方法要说明目标、内容、学生基础和使用步骤。
- 🔴 讲授不等于灌输，演示也不能占满课堂而不给学生练习。
### 回忆卡 2｜任务、项目、探究与合作

先答：四种方法最关键的区别是什么？


- 🔵 任务围绕明确成果；项目处理持续复杂问题；探究靠证据形成解释；合作要求共同目标和个体责任。
- 🔴 比较按目标、周期、活动方式、教师支持和评价证据写。
- ⚫ 做作品不必然是项目，把学生分组不等于合作学习。
### 回忆卡 3｜上机课与差异化

先答：上机课基本流程是什么？怎样照顾差异？


- 🔵 设备检查 → 目标规则 → 任务/演示 → 实践巡视 → 保存提交 → 展示评价 → 安全退出。
- 🔴 设基础、提高、拓展任务；困难者给步骤卡，高水平者给开放任务。
- ⚫ 学生主体不等于教师不指导，设备故障也不能转成无目标活动。
## 四、媒体与教学评价

### 回忆卡 1｜媒体选择

先答：选择教学媒体应考虑哪些因素？


- ⚫ 目标、内容、学习者、方法、环境、成本、易用性、无障碍、安全和版权。
- 🔵 简答写“先服从教学需要，再比较媒体优势和条件”。
- 🔴 媒体越新、动画越多不等于教学效果越好。
### 回忆卡 2｜数字化教学形态

先答：微课、翻转课堂和混合式学习不能被简化为什么？


- 🔵 微课不只一段视频；翻转不只课前看视频；混合不只是线上线下简单相加。
- 🔴 比较时写知识学习时间、课堂活动、线上线下关系和反馈方式。
- ⚫ 使用平台只是形式，是否有效仍看目标、活动和评价。
### 回忆卡 3｜评价分类与质量

先答：诊断、形成、总结性评价各在何时使用？操作技能为什么要用表现性任务和量规？


- ⚫ 教学前查起点是诊断，教学中反馈改进是形成，阶段结束判定是总结；操作目标应看上机任务、作品和过程，量规把功能、方法、规范等目标变成可观察标准。
- 🔵 先由目标确定证据，再用量规描述等级；信度看结果稳定，效度看是否真正测到目标。
- 🔴 量规不能自动消除主观性；只看作品外观可能偏离目标，信度高也不等于效度一定高。
## 五、基础答题与判断纠错

### 回忆卡 1｜特点与区别题

先答：特点题和方法比较题怎样写？


- ⚫ 特点题“定义＋分点解释”；比较题“各自定义＋适用内容＋关键区别”。
- 🔵 每点先写关键词，再说明其具体含义或表现。
- 🔴 只写“更好、更灵活、能培养能力”缺少可评分内容。
### 回忆卡 2｜教学设计与案例题

先答：教学设计和案例分析的基础答题骨架是什么？


- 🔵 设计写学情内容、目标、活动方法、资源、评价和安全；案例按“目标学情 → 问题 → 作用机制 → 对应改进 → 证据验证”。
- 🔴 连续演示使学生跟不上时，可分段示范、即时练习、步骤支架、巡视反馈，再用变式任务检验迁移。
- ⚫ 案例没有脱离条件的唯一教学法；工具不能先于目标决定，只评价作品外观也不能证明学习达成。
### 回忆卡 3｜高频判断陷阱

先答：为什么“学生主体=教师不讲、分组=合作、使用视频=信息化效果好”都错？

- ⚫ 教师仍需组织支持；合作还要共同目标与个体责任；媒体必须服务目标和学习活动。
- 🔵 判断时找题干省略的必要条件，并用一句正确表述替换错误句。
- 🔴 不能因为出现绝对化词就机械判错，仍要说明知识依据。
