# 信息技术与教学论 · 三色回忆卡

> 先答后核对。义务教育按“信息科技”、高中按“信息技术”表述；课标、教材和模型名称以题干指定版本为准。

## 一、课程理念与学习理论

### 回忆卡 1｜课程与核心素养

先答：信息科技为什么不是软件操作课？四项核心素养是什么？

- 🔴 课程同时学习数据、算法、网络和系统原理，并发展解决问题与社会责任；核心素养是信息意识、计算思维、数字化学习与创新、信息社会责任。
- 🔵 简答按“原理学习—问题解决—创新实践—责任伦理”组织。
- ⚫ 计算思维不等于只会编程，使用设备也不等于形成数字化学习能力。

### 回忆卡 2｜主要学习理论

先答：行为主义、认知主义、建构主义、人本主义各关注什么？

- 🔴 依次关注刺激反应与强化、信息加工与知识结构、情境协作中的意义建构、人的需要与自我实现。
- 🔵 匹配：操作训练—行为；结构理解—认知；真实项目—建构；情感选择—人本。
- ⚫ “学生主体”不等于教师退出，发现学习也不等于不给指导。

### 回忆卡 3｜支架与情境

先答：最近发展区、支架和情境学习之间有什么联系？

- 🔴 最近发展区是独立水平与帮助下可达水平之差；支架支持跨越差距并随能力提高逐步撤除。
- 🔵 支架可用步骤卡、示例、提问、同伴协作和即时反馈。
- ⚫ 永久代做会形成依赖，不属于有效支架。

## 二、教学设计与目标

### 回忆卡 1｜系统设计流程

先答：不背模型名称时，教学设计至少包含哪些环节？

- 🔴 分析需求/学习者/内容 → 定目标 → 设计活动、资源与评价 → 实施 → 反馈修订。
- 🔵 ADDIE 是分析、设计、开发、实施、评价；逆向设计先定结果，再定证据，最后设计活动。
- ⚫ 模型名称可能不同，但不能缺目标、学习活动、评价和修订闭环。

### 回忆卡 2｜学情、重点与难点

先答：学情分析要找哪些证据？重点为什么不一定是难点？

- 🔴 学情包括先备知识、认知特点、动机经验、个体差异和设备条件；重点来自课标与内容价值，难点取决于内容和具体学生。
- 🔵 证据用前测、访谈、观察、作品和学习数据，不只凭教师印象。
- ⚫ “教师觉得重要”不能单独证明是重点或难点。

### 回忆卡 3｜可评价目标

先答：怎样把“掌握某知识”改写成可评价目标？

- 🔴 写清学习者、可观察行为、完成条件和达成标准。
- 🔵 句式：“在给定……条件下，学生能解释/比较/设计/调试……，达到……标准。”
- ⚫ 只写了解、掌握、提高、培养，没有行为证据，无法直接评价。

## 三、策略、课堂与媒体

### 回忆卡 1｜任务、项目、探究与合作

先答：四种策略的最小区别是什么？

- 🔴 任务驱动围绕明确成果；项目式处理持续的真实复杂问题；探究用证据形成解释；合作学习要求共同目标和个体责任。
- 🔵 选择时说明目标、任务复杂度、时间、支架和评价证据。
- ⚫ 做作品不必然是项目，围坐一起不等于合作，照步骤操作不等于任务驱动。

### 回忆卡 2｜上机课与差异化

先答：上机课怎样兼顾学习、设备和安全？

- 🔴 检查设备账号 → 明确目标与规则 → 分段示范/发布任务 → 实践巡视 → 保存提交 → 展示评价 → 安全退出。
- 🔵 分层设置基础、进阶、挑战任务；困难者给步骤卡/半成品，高水平者给开放拓展。
- ⚫ 故障时应启用备用方案，不能转成无目标自由活动。

### 回忆卡 3｜媒体与数字化形态

先答：媒体选择首先看什么？微课、翻转和混合学习不能被简化为什么？

- 🔴 媒体先服从目标、内容、学习者、策略和环境；微课不只是一段视频，翻转不只课前看视频，混合不只线上线下相加。
- 🔵 检查表：表征优势、认知负荷、交互反馈、无障碍、技术条件、成本、版权安全。
- ⚫ 动画、声音和平台越多不代表教学效果越好。

## 四、教学评价与反馈

### 回忆卡 1｜评价分类

先答：诊断性、形成性、总结性评价分别在何时、为何使用？

- 🔴 教学前查起点是诊断性；教学中反馈改进是形成性；阶段结束判定达成是总结性。
- 🔵 再按参照分标准、常模、个体内；按主体分自评、互评、教师和其他评价。
- ⚫ 同一测验因使用目的不同，可以属于不同评价类型。

### 回忆卡 2｜量规与测验质量

先答：信度、效度、难度、区分度分别回答什么？

- 🔴 信度看稳定一致，效度看是否测到目标；难度反映难易，区分度反映区分不同水平的能力。
- 🔵 量规应让维度对应目标，水平描述可观察、可区分，并附表现证据。
- ⚫ 信度高是效度高的必要非充分条件；难度值方向先看题目公式。

### 回忆卡 3｜有效反馈

先答：反馈怎样回答“现在怎样、差在哪里、下一步做什么”？

- 🔴 有效反馈及时、具体，面向任务过程和可执行改进，而非评价人格。
- 🔵 写法：指出证据 → 对照目标 → 定位问题 → 给下一步策略 → 安排再次表现机会。
- ⚫ “很好、继续努力”没有指出差距和行动，信息量不足。

## 五、案例、教学设计与教研

### 回忆卡 1｜案例分析

先答：面对一段课堂案例，如何避免只写“好/不好”？

- 🔴 先判断做法，再引用课标/理论，分析对学习的影响，最后给具体改进和评价证据。
- 🔵 高分链：“观察证据 → 理论解释 → 学习影响 → 可执行修改 → 如何验证”。
- ⚫ “激发兴趣、培养能力”若无行为和证据，通常不能形成完整分析。

### 回忆卡 2｜教学设计大题

先答：一份可评分的教学设计应包含哪些互相对应的部分？

- 🔴 课标学情、目标、重点难点、情境任务、活动流程、资源支架、评价、安全伦理和反思。
- 🔵 逐项检查“目标—活动—评价”是否一一对应，即教—学—评一致。
- ⚫ 技术工具不能先于目标决定，作品外观也不能替代学习证据。

### 回忆卡 3｜说课、评课与行动研究

先答：说课、评课和行动研究分别输出什么？

- 🔴 说课解释教什么、为何和怎样教评；评课用观察证据判断目标达成；行动研究按计划—行动—观察—反思循环改进。
- 🔵 评课表述仍按“证据 → 解释 → 影响 → 建议”。
- ⚫ 教案是实施方案，说课是对方案依据的解释，二者不能等同。
