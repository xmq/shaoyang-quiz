# 信息技术与教学论：三色笔记

三色按考频和出题稳定性划分：红色优先背熟，蓝色用于补齐得分点，黑色用于防漏和边界判断。

## 🔴 高频考点

- 🔴 **教育技术基本概念**：教育技术是利用技术促进学习过程和资源设计、开发、利用、管理与评价。 常见考法：判断教育技术研究对象。 易错边界：教育技术不等于单纯会用设备。
- 🔴 **教学设计**：分析学习者、目标、内容、策略、媒体和评价。 常见考法：排序教学设计流程。 易错边界：目标要可观察、可评价。
- 🔴 **教学目标**：知识与技能、过程与方法、情感态度价值观常用于目标表达。 常见考法：判断目标层次和表述是否清楚。 易错边界：“了解”比“掌握”要求低。
- 🔴 **教学媒体选择**：媒体要服务目标、内容和学习者。 常见考法：按场景选择视频、动画、课件、实验。 易错边界：技术新不代表教学效果一定好。
- 🔴 **信息化教学模式**：翻转课堂、混合学习、项目学习等强调学习活动设计。 常见考法：判断模式特点。 易错边界：翻转课堂不是把视频发给学生就结束。
- 🔴 **教学评价**：形成性评价关注过程，终结性评价关注结果。 常见考法：判断评价类型。 易错边界：评价不只等于考试分数。

## 🔵 中频考点

- 🔵 **学习理论**：行为主义、认知主义、建构主义关注点不同。 常见考法：按教学活动判断理论依据。 易错边界：建构主义强调学习者主动建构。
- 🔵 **课件设计**：界面清晰、重点突出、交互适度。 常见考法：判断课件优劣。 易错边界：堆动画和文字会降低学习效率。
- 🔵 **资源建设**：数字资源应可获取、可复用、适配教学目标。 常见考法：判断资源类型和作用。 易错边界：资源多不代表课程好。
- 🔵 **课堂互动**：提问、讨论、投票、协作工具提升参与。 常见考法：判断互动方式。 易错边界：互动要围绕学习目标。
- 🔵 **教师信息素养**：包括技术应用、资源评价、伦理安全和教学融合。 常见考法：判断教师能力要求。 易错边界：会操作软件只是其中一部分。

## ⚫ 低频了解

- ⚫ **教育数据**：学习数据可用于诊断和改进教学。 常见考法：概念判断。 易错边界：使用数据要注意隐私。
- ⚫ **智慧教育**：依托平台、数据和智能技术支持个性化学习。 常见考法：概念判断。 易错边界：智慧教育不是设备堆砌。
- ⚫ **在线学习支持**：学习平台、论坛、测验和反馈共同支撑学习。 常见考法：判断平台功能。 易错边界：在线学习更需要过程管理。

## 考前框架与简答

### 必会框架

- **教学设计主线**：学习者与需求分析 → 确定可评价目标 → 组织内容与活动 → 选择媒体和策略 → 实施 → 评价与修正。每一步都服务教学目标，技术工具不是起点。
- **目标表述**：通常写清“谁在什么条件下完成什么可观察行为，达到何种程度”。“理解、掌握”过于笼统，最好落到“能说明、能比较、能完成”等行为。
- **评价关系**：诊断性评价用于教学前摸底，形成性评价用于教学中改进，终结性评价用于阶段结束后判断达成程度。

### 常见简答题

- **教学媒体选择应遵循什么原则？** 以目标和内容为核心，适合学习者特征与教学环境，兼顾效果、成本、可获得性和可操作性，并通过评价验证实际效果。
- **翻转课堂为什么不等于课前看视频？** 翻转课堂还要求课前自主学习、课堂中的问题解决与协作活动、及时反馈和学习评价；视频只是资源之一。
