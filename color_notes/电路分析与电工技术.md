# 电路分析与电工技术 · 三色回忆卡

> 先答后核对。🔴是必须写出的结论与基本公式；🔵是计算或作答顺序；⚫是成立条件、单位和高频陷阱。完整推导与电路图见“讲义”。

## 一、直流电路基础

### 回忆卡 1｜参考方向与功率

先答：关联参考方向下怎样判断元件吸收还是发出功率？开路与短路分别固定什么量？

- 🔴 `i=dq/dt`，关联参考方向下 `p=ui`；`p>0` 吸收功率，`p<0` 发出功率。开路电流为0，短路电压为0。
- 🔵 先标电压、电流参考方向，再代数值判功率正负；负值只说明实际方向与参考方向相反。
- ⚫ 开路电压、短路电流都不一定为0；计算前统一V、A、Ω、F、H、Hz及数量级。

### 回忆卡 2｜欧姆定律、KCL与KVL

先答：欧姆定律、KCL、KVL各写什么？列方程时怎样避免符号错误？

- 🔴 电阻 `u=Ri`、`G=1/R`；KCL为任一节点电流代数和0；KVL为任一闭合回路电压代数和0。
- 🔵 标参考方向 → 统一流入/流出符号或回路绕行方向 → 写方程 → 求解并解释负号。
- ⚫ KCL依据电荷守恒，KVL依据能量守恒；不能算到一半更换参考方向。

### 回忆卡 3｜串并联、分压与分流

先答：串联、并联怎样判定并求等效？两电阻分压和分流公式是什么？

- 🔴 串联 `Req=ΣR`、`Uk=U·Rk/(R1+R2)`；并联 `1/Req=Σ1/Rk`、`I1=I·R2/(R1+R2)`；相同电阻并联 `Req=R/n`。
- 🔵 先按节点关系判串并联，再求等效；用“串联大于任一电阻、并联小于最小电阻”验算。
- ⚫ 串联中间节点不能有分支；并联必须连接同一对节点，不能按图形位置判断。

### 回忆卡 4｜电源模型与等效变换

先答：理想电压源、理想电流源分别固定什么？实际源怎样互换？

- 🔴 理想电压源固定端电压，理想电流源固定支路电流；`Us`串联`Rs` ↔ `Is=Us/Rs`并联同一`Rs`。
- 🔵 变换时保持端口、源方向和电阻数值对应，再检查开路电压与短路电流是否一致。
- ⚫ 等效只对外端口成立；理想电压源不能单独等效为理想电流源。

## 二、线性电路分析与网络定理

### 回忆卡 5｜节点电压法与网孔电流法

先答：节点法、网孔法各选什么未知量？遇到非参考节点间电压源怎么办？

- 🔴 节点法以节点电压列KCL；网孔法以平面网孔电流列KVL，共享电阻压降写`R(Ik−Ij)`；跨两非参考节点的电压源构成超级节点。
- 🔵 选参考节点/网孔 → 标未知量 → 支路量用未知量表示 → 列独立方程 → 联立并验算。
- ⚫ 超级节点除整体KCL外还要补电压约束；网孔法只直接适用于平面电路。

### 回忆卡 6｜叠加、戴维宁与诺顿

先答：三项网络定理怎样使用？怎样求`Uth、Rth、In`与最大功率？

- 🔴 叠加用于线性电压/电流；戴维宁为`Uth`串`Rth`，诺顿为`In`并`Rth`，`Uth=InRth`；纯电阻直流最大功率时`RL=Rth`，`Pmax=Uth²/(4Rth)`。
- 🔵 叠加时电压源短路、电流源开路、受控源保留；`Uth=Uoc`，`In=Isc`；关独立源向端口看求`Rth`，含受控源则加测试源求`Utest/Itest`。
- ⚫ 功率不能直接叠加；端口等效要求线性网络，最大功率条件不能无条件套到交流复阻抗或非线性电路。

## 三、一阶动态电路

### 回忆卡 7｜电容、电感与换路定律

先答：电容和电感的伏安关系、储能、直流稳态等效与连续量是什么？

- 🔴 `iC=C·duC/dt，WC=CuC²/2`，直流稳态开路且`uC`连续；`uL=L·diL/dt，WL=LiL²/2`，直流稳态短路且`iL`连续。
- 🔵 先求换路前稳态的`uC(0−)`或`iL(0−)`，用连续性得到`0+`初值，再分析换路后电路。
- ⚫ 连续性以无冲激为前提；电容电流和电感电压可在换路瞬间突变。

### 回忆卡 8｜一阶电路三要素法

先答：一阶响应统一式和时间常数是什么？三要素按什么顺序求？

- 🔴 `f(t)=f(∞)+[f(0+)−f(∞)]e^(−t/τ)`；RC中`τ=ReqC`，RL中`τ=L/Req`。
- 🔵 求换路前稳态得`f(0+)` → 求换路后最终稳态`f(∞)` → 关闭独立源求储能元件所见`Req` → 代统一式。
- ⚫ `t=τ`只完成总变化约63.2%，约`5τ`才可近似新稳态；求`Req`时受控源不能关闭。

## 四、正弦稳态交流电路

### 回忆卡 9｜正弦量、有效值与相量

先答：正弦量三要素是什么？最大值、有效值、频率与角频率怎样换算？

- 🔴 三要素为最大值、角频率、初相位；`ω=2πf，T=1/f`，正弦有效值`U=Um/√2，I=Im/√2`。
- 🔵 先统一为同频正弦量，再用有效值相量进行复数运算，最后按题目要求还原幅值和相位。
- ⚫ 相量不是瞬时量，不能含时间变量；额定值和交流电表读数通常为有效值。

### 回忆卡 10｜阻抗、相位与交流功率

先答：R、L、C阻抗及相位关系是什么？串联RLC和三种功率怎样计算？

- 🔴 `ZR=R，ZL=jωL，ZC=1/(jωC)`；串联RLC有`Z=R+j(XL−XC)`；`P=UIcosφ，Q=UIsinφ，S=UI`且`S²=P²+Q²`。
- 🔵 算`XL=2πfL、XC=1/(2πfC)` → 合成复阻抗 → 求电流及相位 → 再求P、Q、S；电阻同相、感性电流滞后、容性电流超前。
- ⚫ `XC`随频率升高而减小；有功W、无功var、视在VA，感性负载通常`Q>0`。

### 回忆卡 11｜串联谐振

先答：串联RLC何时谐振？谐振时阻抗、电流和功率因数怎样变化？

- 🔴 `XL=XC`时谐振，`ω0=1/√(LC)`；总阻抗为R且最小，电流最大，功率因数为1。
- 🔵 先令虚部`XL−XC=0`求谐振频率，再代阻抗、电流或元件端电压公式。
- ⚫ 电源端无功功率为0不表示L、C不工作；元件端电压可能远大于电源电压。

## 五、三相、电工控制与可靠性

### 回忆卡 12｜对称三相电路

先答：Y接、Δ接的线相量关系及三相有功功率公式是什么？

- 🔴 Y接`UL=√3Up，IL=Ip`；Δ接`UL=Up，IL=√3Ip`；总有功`P=√3ULILcosφ`。
- 🔵 先判接法并完成线量/相量换算，再按每相或线量公式计算，最后乘三相总量。
- ⚫ `√3ULILcosφ`以三相对称为前提；线电压与相电压还存在30°相位关系。

### 回忆卡 13｜变压器与异步电动机

先答：理想变压器的电压、电流、阻抗关系是什么？同步转速和转差率怎样算？

- 🔴 `U1/U2=N1/N2，I1/I2=N2/N1，Zin=(N1/N2)²ZL`；`n1=60f/p，s=(n1−n)/n1`。
- 🔵 先由匝数比换算电压与电流，再做阻抗折算；电机题先由频率和磁极对数求同步转速，再求转差率。
- ⚫ 正常异步电动运行通常`0<s<1`且转速略低于同步转速；四极表示磁极对数`p=2`。

### 回忆卡 14｜继电接触控制

先答：自锁、互锁怎样接？熔断器、热继电器和接触器分别保护什么？

- 🔴 自锁用接触器常开辅助触点并联启动按钮；互锁用双方常闭辅助触点交叉串入对方线圈。
- 🔵 控制图按“启动—保持—停止—互锁—保护”检查电流路径和触点常态。
- ⚫ 熔断器/断路器主要防短路，热继电器防长期过载，接触器失电释放形成失压保护，三者不能互相替代。

### 回忆卡 15｜系统可靠性

先答：功能串联和功能并联的可靠度怎样计算？两个可信度0.9的串联系统是多少？

- 🔴 独立元件功能串联`Rs=ΠRi`，功能并联`Rp=1−Π(1−Ri)`；两个0.9串联为`0.9²=0.81`。
- 🔵 先按“任一失效即失败/至少一个成功即成功”判逻辑结构，再代可靠度或先算全部失败概率。
- ⚫ 公式要求失效相互独立；可靠性逻辑串并联不是电阻物理串并联，不使用分压分流公式。
