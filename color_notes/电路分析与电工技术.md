# 电路分析与电工技术｜三色回忆卡

先口头回答，再点开核对：红色写结论，蓝色写原因或计算依据，黑色写成立边界与易错点。

## 一、电路与基本物理量

### 回忆卡1：电路由哪些部分组成

先答：一个完整电路通常由哪些部分组成，各有什么作用？

- 🔴 电源、负载、导线、控制和保护装置。
- 🔵 按“供能—用能—连接—控制保护”的顺序回答。
- ⚫ 开关负责控制通断；熔断器、断路器等负责异常保护。

### 回忆卡2：开路和短路

先答：开路和短路的电流、电压特点分别是什么？

- 🔴 开路电流为0；短路两端电压近似为0。
- 🔵 判断时分别抓住“通路断开”和“近零电阻连接”。
- ⚫ 开路电压不一定为0；短路电流也不一定为0，且可能很大。

### 回忆卡3：欧姆定律和功率

先答：欧姆定律和电功率的基本公式是什么？

- 🔴 `U=IR`；`P=UI=I²R=U²/R`。
- 🔵 先统一V、A、Ω、W等单位，再选择已知量最多的公式。
- ⚫ 不能断言所有非线性元件在任何状态下都满足欧姆定律。

## 二、串并联与基本定律

### 回忆卡4：串联和并联的特点

先答：串联、并联电路分别有哪些基本特点？

- 🔴 串联电流相同、电压相加、电阻相加；并联电压相同、电流相加、倒数相加。
- 🔵 简答按“电流—电压—等效电阻”三个角度逐点对比。
- ⚫ 并联总电阻小于任一支路电阻，串联总电阻大于任一串联电阻。

### 回忆卡5：三个相同电阻并联

先答：三个 `6 Ω` 电阻并联后的等效电阻是多少？

- 🔴 `R总=R/n=6/3=2 Ω`。
- 🔵 相同电阻并联直接用 `R/n`，再用“结果小于6 Ω”检查。
- ⚫ `R/n` 只适用于阻值相同的并联电阻，不适用于一般并联。

### 回忆卡6：KCL与KVL

先答：KCL和KVL分别说明什么？

- 🔴 KCL：节点流入电流等于流出电流；KVL：闭合回路电压代数和为0。
- 🔵 KCL抓“节点与电流”，KVL抓“回路与电压”。
- ⚫ 两条定律都要先规定参考方向，负值表示实际方向与假设相反。

## 三、电阻、电容和电感

### 回忆卡7：电容与容抗

先答：电容的基本作用是什么，容抗怎样随频率和电容量变化？

- 🔴 电容储存电场能量；`XC=1/(2πfC)`，频率或电容量越大，容抗越小。
- 🔵 一步计算先把 `μF` 换成 `F`，再直接代公式。
- ⚫ “隔直通交”是概括；稳定直流下理想电容相当于开路。

### 回忆卡8：电感与感抗

先答：电感的基本作用是什么，感抗怎样变化？

- 🔴 电感储存磁场能量；`XL=2πfL`，频率或电感量越大，感抗越大。
- 🔵 与电容对照记：容抗随频率下降，感抗随频率上升。
- ⚫ “通直阻交”按理想稳态理解；实际线圈仍有电阻和损耗。

## 四、交流与电工控制

### 回忆卡9：正弦交流三要素

先答：正弦交流电的三要素是什么？

- 🔴 最大值、角频率（或频率）、初相位。
- 🔵 看到 `u=Um sin(ωt+φ)`，依次找 `Um、ω、φ`。
- ⚫ 日常所说的220 V市电通常指有效值，不是最大值。

### 回忆卡10：R、L、C相位关系

先答：纯电阻、纯电感、纯电容的电压电流相位关系是什么？

- 🔴 电阻同相；电感中电流滞后电压；电容中电流超前电压。
- 🔵 口诀：“阻同、感流后、容流先”。
- ⚫ 串联RLC谐振时总电路呈电阻性，功率因数为1。

### 回忆卡11：交流功率与功率因数

先答：有功、无功、视在功率分别表示什么，提高功率因数有什么意义？

- 🔴 有功功率真正做有用功，无功功率在电源与储能元件间交换，视在功率表示总供电容量；功率因数是 `P/S`。
- 🔵 提高功率因数可减小同样有功功率下的线路电流和损耗，降低电压损失，提高设备容量利用率。
- ⚫ 无功功率不直接做有用功，但建立电磁场需要它；补偿过多可能使系统呈容性。

### 回忆卡12：变压器与电表接法

先答：变压器有什么基本作用，电压表和电流表应怎样接？

- 🔴 变压器用于交流变压、电气隔离和阻抗变换；电压表并联，电流表串联。
- 🔵 电压表内阻应大以少分流，电流表内阻应小以少产生附加压降。
- ⚫ 普通变压器不能直接变换恒定直流；电流表不能直接并接电源两端。

### 回忆卡13：三相交流与异步电动机

先答：三相交流有什么特点，异步电动机由什么组成？

- 🔴 三相电动势同频、等幅、相位互差120°；电动机主要由定子和转子组成。
- 🔵 简答补充：三相电形成旋转磁场，感应转矩带动转子。
- ⚫ 正常运行时转子转速低于同步转速，所以称“异步”。

### 回忆卡14：熔断器、热继电器和接触器

先答：三种低压电器分别有什么主要作用？

- 🔴 熔断器防短路，热继电器防长期过载，接触器频繁控制主电路通断。
- 🔵 按“元件名称＋主要作用”逐条回答即可得分。
- ⚫ 热继电器不能替代短路保护，接触器本身也不是完整保护装置。

## 五、可靠性与安全判断

### 回忆卡15：串联系统可靠度

先答：两个可靠度均为0.9的独立元件串联系统，可靠度是多少？

- 🔴 `0.9×0.9=0.81`。
- 🔵 串联系统必须全部正常，所以独立可靠度相乘。
- ⚫ 这是功能可靠性的串联逻辑，不是电阻串联值的计算。

### 回忆卡16：安全用电

先答：安全用电常见措施有哪些？

- 🔴 正确接地、绝缘完好、配置保护、检修前断电验电、避免湿手操作。
- 🔵 简答用“工程保护＋个人操作”两类组织答案。
- ⚫ 不能用粗铜丝代替保险丝，也不能只断开零线就开始检修。

### 回忆卡17：直接套公式怎样不丢步骤

先答：串并联、容抗和交流功率的一步计算应按什么顺序书写？

- 🔴 判断关系→写公式→统一单位→代入计算→检查结论。
- 🔵 串联分压 `U1=UR1/(R1+R2)`，容抗 `XC=1/(2πfC)`，单相有功功率 `P=UIcosφ`。
- ⚫ 分压只适用于确为串联的电阻；`U、I` 取有效值；`μF` 必须先换成 `F`。

### 回忆卡18：功率因数为什么影响损耗

先答：同样有功功率下，提高功率因数为什么通常能降低线路损耗？

- 🔴 因 `I=P/(Ucosφ)`，功率因数提高使电流减小，`I²R` 线路损耗随之降低。
- 🔵 得分链条写“功率、电压不变→电流减小→线路损耗和压降减小”。
- ⚫ 并联电容补偿不是增加负载有功功率，也不是越多越好，过补偿会使系统呈容性。

## 六、仪表、一阶响应与排障

### 回忆卡19：万用表三种接法

先答：测电压、电流、电阻时分别怎样接，最重要的安全边界是什么？

- 🔴 电压并联、电流串联、电阻挡断电测量。
- 🔵 先确认交直流、量程、插孔和表笔，再接入；未知量先用较大量程。
- ⚫ 电流挡不能跨接电源，电阻挡不能测带电电路，电容还应确认安全放电。

### 回忆卡20：示波器怎样读幅值和频率

先答：怎样由屏幕格数求峰峰值、周期和频率？

- 🔴 `Upp=垂直格数×V/div×探头倍率`；`T=水平格数×s/div`；`f=1/T`。
- 🔵 先读刻度和探头倍率，再数格；波形不稳时检查触发源和触发电平。
- ⚫ 普通台式示波器地夹常接保护地，不能随意夹到高电位点以免短路。

### 回忆卡21：RC、RL时间常数

先答：RC和RL一阶电路的时间常数是什么，一个时间常数表示什么？

- 🔴 `τRC=RC`，`τRL=L/R`；充电一个τ约完成最终变化量的63.2%。
- 🔵 判断顺序是初值、终值、时间常数；τ越大，响应通常越慢。
- ⚫ 电容电压和电感电流一般不能突变；复杂多储能网络不能机械套单一τ。

### 回忆卡22：基础排障顺序

先答：设备无输出时为什么要按信号路径逐段检查？

- 🔴 先安全观察，再查电源和保护，随后从输入到负载逐节点比较，逐步缩小故障范围。
- 🔵 万用表看直流状态和通断，示波器看各级波形；从最后正常节点与第一个异常节点之间定位。
- ⚫ 保险反复熔断不能换更大额定值掩盖故障，应先排查短路或过载。
