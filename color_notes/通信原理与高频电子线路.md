# 📕 通信原理与高频电子线路 · 三色笔记

使用说明：🔴必背概念与判据；🔵公式、单位和计算步骤；⚫成立条件与易错点。通信原理与高频线路在不同学校可能分为两门课，本笔记覆盖共同主干。

## 一、通信系统与信息量

### 1. 系统模型与传输方式

🔴【基本链路】：信源 → 发送设备 → 信道+噪声 → 接收设备 → 信宿。

🔴【发送设备】：编码、调制、放大，使消息变为适合信道传输的信号；接收端完成逆过程。

🔴【单工】：只能单向；【半双工】：双向但不能同时，如对讲机；【全双工】：双向同时，如电话。

🔴【模拟/数字通信】：按传输信号取值连续或离散划分，不等同于信源本身是否模拟。

### 2. 信息量、码元与速率

🔴【自信息量】：`I=−log2P(x)` bit；等概率M进制每码元携带 `log2M` bit。

🔴【码元速率】：`RB=1/TB`，单位Bd；【比特率】：单位bit/s。

🔵【关系】：理想等概率M进制 `Rb=RBlog2M`；只有二进制时数值才常相等。

⚫【陷阱】：波特率不是“每秒比特数”，也不宜简单写成“波形变化次数”。

### 3. 信噪比与香农容量

🔴【分贝】：功率比 `SNRdB=10log10(S/N)`；电压比在同阻抗下可用 `20log10(U2/U1)`。

🔴【香农公式】：`C=B log2(1+S/N)` bit/s。

🔵【计算步骤】：dB先转功率比 `S/N=10^(dB/10)` → 带宽换Hz → 代公式 → 结果写bit/s或kbit/s。

⚫【物理意义】：C是给定带宽和信噪比下可靠通信的理论上限；提高S/N的收益呈对数增长。

## 二、信道、噪声与基带传输

### 4. 信道与噪声

🔴【信道】：有线/无线、恒参/随参；常见影响为衰减、失真、时延和噪声。

🔴【白噪声】：功率谱密度在研究频段内近似常数；高斯白噪声的幅度服从高斯分布。

🔴【误码率】：错误比特数/传输总比特数；不能与误码个数混为单位。

### 5. 无码间串扰与眼图

🔴【奈奎斯特第一准则】：接收抽样时刻本码元响应不为0、其他码元响应为0，可无码间串扰判决。

🔴【理想低通信道最高码元率】：`RB,max=2B` Bd；若M进制，理想比特率上限 `2Blog2M`。

🔴【升余弦】：带宽 `B=(1+α)RB/2`，`0≤α≤1`；α越大，时域拖尾更小但占用带宽增大。

🔴【眼图】：眼睛张开越大，码间串扰与噪声通常越小；最佳抽样时刻在眼睛最张开处。

## 三、抽样、量化与PCM

### 6. 抽样定理

🔴【低通信号】：最高频率fm，理想无混叠要求 `fs≥2fm`。

⚫【工程条件】：实际抗混叠滤波器有过渡带，抽样率常留裕量；等号是理想下限。

### 7. PCM

🔴【流程】：抽样 → 量化 → 编码；接收端译码、低通重建。

🔴【码率】：每秒fs个样值、每样值n bit，则 `Rb=fsn`；多路复用还要乘路数并考虑帧开销。

🔴【量化级】：n位均匀量化有 `L=2^n` 级；量化间隔约为满量程/L，误差绝对值通常≤半个量化间隔。

🔴【非均匀量化】：A律/μ律对小信号分得更细，改善小信号量化信噪比。

⚫【13折线A律】：8位码常由极性、段落码和段内码构成；具体码字规则以教材表格为准。

## 四、模拟调制

### 8. AM、DSB-SC与SSB

🔴【AM频谱】：载波fc及上下边带 `fc±fm`；基带最高频率fm时带宽 `BAM=2fm`。

🔴【调幅系数】：单音AM为避免过调幅通常 `0≤ma≤1`；`ma>1` 包络检波会失真。

🔴【DSB-SC】：抑制载波、保留双边带，带宽2fm，需相干解调。

🔴【SSB】：只传一个边带，带宽fm，功率和频带利用率高，但收发实现更复杂。

🔵【相干解调】：乘以同频同相本振 → 低通滤波；频率或相位不同步会衰减、失真。

### 9. FM与PM

🔴【FM】：幅度理想不变，瞬时频率随调制信号变化；抗幅度噪声能力较好。

🔴【调频指数】：单音FM `β=Δf/fm`；卡森公式 `BFM≈2(Δf+fm)=2(β+1)fm`。

⚫【窄带/宽带】：β≪1为窄带近似，β较大为宽带；FM带宽不能一律写成2fm。

## 五、数字调制与差错控制

### 10. ASK、FSK、PSK与QAM

🔴【2ASK】：用幅度表示比特，易受幅度噪声影响。【2FSK】：用两种频率表示。【2PSK/BPSK】：用相差π的载波表示。

🔴【QPSK】：4种相位，每码元2 bit；M进制每码元 `log2M` bit。

🔴【QAM】：同时利用幅度和相位，提高频谱效率，但阶数越高对信噪比和线性度要求越高。

🔴【相干BPSK误码率】：常见AWGN条件下与 `Eb/N0` 有关；比较性能时必须统一检测方式和每比特能量。

⚫【差分编码】：可解决PSK相干载波的相位模糊，但性能和实现代价需按题目模型判断。

### 11. 信道编码

🔴【目的】：加入结构化冗余用于检错/纠错，以带宽或码率换可靠性。

🔴【奇偶校验】：能检出奇数个比特错误，不能定位并纠正。【循环码】：利用生成多项式，系统码常由信息位+校验位组成。

🔴【汉明距离】：要检出e个错误需 `dmin≥e+1`；纠正t个错误需 `dmin≥2t+1`。

## 六、复用、移动与光纤通信

### 12. 复用与多址

🔴【FDM】：不同频带同时传输。【TDM】：不同时间隙。【WDM】：光波长复用。【CDM】：不同码序列。

⚫【多址与复用】：技术思想相关，但多址强调多个用户共享资源，复用强调多路信号合并到一条链路。

### 13. 移动通信代际

🔴【主线】：1G模拟语音；2G数字语音与短信；3G移动数据；4G全IP移动宽带；5G增强移动宽带、海量连接、低时延高可靠场景。

⚫【考试边界】：代际标准众多，避免把某一地区制式说成全球唯一；以课程给定标准为准。

### 14. 电磁波与光纤

🔴【频率由低到高】：无线电/微波 → 红外 → 可见光 → 紫外 → X射线 → γ射线；波长顺序相反。

🔴【光纤窗口】：常用 `1.31 μm` 与 `1.55 μm`；1.55 μm附近损耗更低，1.31 μm附近色散特性具有传统优势。

🔴【数值孔径】：反映光纤集光能力；单模光纤带宽高、色散小、适合长距离高速传输。

## 七、高频电子线路

### 15. 高频小信号放大器

🔴【调谐回路】：完成选频、阻抗变换和负载作用；品质因数Q越高，选择性通常越好、通频带越窄。

🔵【带宽近似】：单调谐回路 `BW≈f0/Q`；多级级联提高选择性但总带宽可能变窄。

### 16. 高频功率放大器

🔴【丙类谐振功放】：导通角小于180°，效率高；集电极电流脉冲经谐振回路选出基波近似正弦。

🔴【工作状态】：欠压、临界、过压按负载线和电流脉冲形状判断；最佳状态常在临界附近，具体以教材图为准。

### 17. 混频与超外差接收

🔴【混频】：非线性器件产生和频、差频等组合频率，选出固定中频 `fIF=|fLO−fRF|`。

🔴【超外差链路】：天线/高放 → 混频+本振 → 中频放大 → 检波/解调 → 低频放大。

🔴【镜像频率】：另一射频也能与本振产生同一中频；提高前端选择性可抑制镜像干扰。

### 18. 反馈振荡器与锁相环

🔴【振荡条件】：`|AF|=1` 且总相移为 `2kπ`；起振时环路增益略大于1，稳幅后回到1。

🔴【PLL】：鉴相器+环路滤波器+压控振荡器；锁定后输出频率/相位跟踪输入，可用于频率合成、解调和载波恢复。
