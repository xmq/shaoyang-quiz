# 通信原理与高频电子线路 · 三色回忆卡

> 先答后核对。🔴是概念、判据和基本公式；🔵是换算、计算和框图步骤；⚫是模型前提、同步条件与易错边界。两门课分开考试时以本校范围为准。

## 一、通信系统与信息量

### 回忆卡 1｜系统模型与传输方式

先答：通信系统基本链路是什么？单工、半双工、全双工怎样区分？

- 🔴 信源 → 发送设备 → 信道+噪声 → 接收设备 → 信宿；单工仅单向，半双工可双向但不能同时，全双工可双向同时。
- 🔵 框图题按“消息产生—编码/调制/放大—信道—解调/译码—接收”标各模块作用。
- ⚫ 对讲机是半双工、电话通常全双工；模拟/数字通信按传输信号取值连续/离散划分，不按信源类型直接判断。

### 回忆卡 2｜信息量、码元与速率

先答：自信息量、码元速率、比特率如何定义？M进制下怎样换算？

- 🔴 `I=−log2P(x)` bit；`RB=1/TB` Bd；等概率M进制每码元`log2M` bit，故`Rb=RBlog2M`。
- 🔵 先判M和码元持续时间，再分别求Bd与bit/s；二进制时两者数值常相等。
- ⚫ 波特率不是“每秒比特数”，也不能笼统定义为波形变化次数。

### 回忆卡 3｜信噪比与香农容量

先答：dB怎样还原功率比？香农公式说明什么上限？

- 🔴 功率信噪比`SNRdB=10log10(S/N)`；香农容量`C=B log2(1+S/N)` bit/s。
- 🔵 dB转线性比`S/N=10^(dB/10)` → 带宽换Hz → 代公式 → 写bit/s或kbit/s。
- ⚫ C是给定带宽和信噪比下可靠通信的理论上限；提高信噪比的容量收益呈对数增长。

## 二、信道、噪声与基带传输

### 回忆卡 4｜信道、噪声与误码率

先答：信道常怎样分类？白噪声和误码率分别怎样表述？

- 🔴 信道可分有线/无线、恒参/随参；白噪声在研究频段内功率谱密度近似常数，高斯白噪声幅度服从高斯分布；误码率=错误比特数/总比特数。
- 🔵 分析链路时依次检查衰减、失真、时延、噪声及它们对判决的影响。
- ⚫ 误码率是无量纲比值，不是错误个数；“白”只说明功率谱，不自动说明幅度分布。

### 回忆卡 5｜无码间串扰与眼图

先答：奈奎斯特第一准则、理想最高码元率和升余弦带宽是什么？眼图怎样判断质量？

- 🔴 抽样时本码元响应非0、其他码元响应为0可无码间串扰；理想低通`RB,max=2B`，升余弦`B=(1+α)RB/2`。
- 🔵 由B求码元率后乘`log2M`得理想比特率；眼图在眼睛最张开处抽样，观察开口高度、宽度和交叉点。
- ⚫ `0≤α≤1`，α增大可减小时域拖尾但占用带宽增大；眼图开口大通常表示噪声和码间串扰较小。

## 三、抽样、量化与PCM

### 回忆卡 6｜抽样定理

先答：低通信号无混叠抽样的最低频率是什么？为什么工程上通常取更高？

- 🔴 最高频率为`fm`的带限低通信号，理想无混叠要求`fs≥2fm`。
- 🔵 先确定信号最高有效频率，再结合抗混叠滤波器过渡带选择实际采样率。
- ⚫ 等号是理想下限；非带限信号或实际滤波器必须留裕量。

### 回忆卡 7｜PCM码率与量化

先答：PCM流程、码率、量化级和误差怎样计算？

- 🔴 PCM为抽样 → 量化 → 编码；每秒`fs`样值、每样值n bit时`Rb=fsn`；n位有`L=2^n`级，均匀量化误差绝对值通常≤半量化间隔。
- 🔵 多路PCM用`fs×每样值位数×路数`，再加帧同步等开销；由满量程除级数求量化间隔。
- ⚫ A律/μ律使小信号量化更细；A律13折线具体码字按教材表格，不能只凭段号猜码。

## 四、模拟调制

### 回忆卡 8｜AM、DSB-SC与SSB

先答：三种调幅信号传哪些频谱分量、占多大带宽、怎样解调？

- 🔴 AM含载波及`fc±fm`边带且`B=2fm`；DSB-SC抑制载波、保留双边带并需相干解调；SSB只留一边带且`B=fm`。
- 🔵 相干解调为乘同频同相本振 → 低通滤波；单音AM通常由包络极值或题给幅度求调幅系数。
- ⚫ AM为避免过调幅通常`0≤ma≤1`；本振频率/相位不同步会造成相干解调衰减或失真。

### 回忆卡 9｜FM与PM

先答：FM改变什么量？调频指数和卡森带宽怎样计算？

- 🔴 FM理想幅度不变、瞬时频率随消息变化；单音`β=Δf/fm`，卡森公式`BFM≈2(Δf+fm)=2(β+1)fm`。
- 🔵 先由最大频偏和最高调制频率求β，再判断窄带/宽带并用题目指定公式求带宽。
- ⚫ `β≪1`才可作窄带近似；FM带宽不能一律写成`2fm`，幅度恒定也不代表不受任何噪声影响。

## 五、数字调制与差错控制

### 回忆卡 10｜ASK、FSK、PSK与QAM

先答：四类数字调制分别改变载波什么参数？M进制每码元携带多少比特？

- 🔴 ASK、FSK、PSK分别用幅度、频率、相位表示信息；QAM联合幅度和相位；M进制每码元`log2M` bit，QPSK每码元2 bit。
- 🔵 比较题统一写频谱效率、抗噪性、同步/检测方式、功放线性要求和实现复杂度。
- ⚫ 高阶QAM频谱效率高但需更高信噪比和线性度；BPSK误码性能比较必须统一`Eb/N0`和检测模型，差分编码仅缓解相位模糊。

### 回忆卡 11｜信道编码与汉明距离

先答：信道编码为何加冗余？最小距离怎样决定检错、纠错能力？

- 🔴 编码以码率/带宽换可靠性；检e错需`dmin≥e+1`，纠t错需`dmin≥2t+1`。
- 🔵 循环码按生成多项式作模2除法得到校验位；接收端用余数/伴随式判断错误。
- ⚫ 单奇偶校验能检奇数个比特错误但不能定位纠正；具体循环码纠错能力仍由码结构和最小距离决定。

## 六、复用、移动与光纤通信

### 回忆卡 12｜复用与多址

先答：FDM、TDM、WDM、CDM分别用什么资源区分各路信号？

- 🔴 FDM分频带、TDM分时隙、WDM分光波长、CDM分码序列。
- 🔵 选择题先看共享维度，再判断是否需要保护带、同步时隙或码相关检测。
- ⚫ 复用强调多路信号合到链路，多址强调多用户共享资源；思想相关但术语不能完全互换。

### 回忆卡 13｜移动通信代际

先答：1G至5G各自的主线能力是什么？

- 🔴 1G模拟语音；2G数字语音和短信；3G移动数据；4G全IP移动宽带；5G覆盖增强宽带、海量连接、低时延高可靠场景。
- 🔵 作答按“信号/网络特征—主要业务—代表能力”概括，不必堆砌所有制式名称。
- ⚫ 各地区和组织标准不同，不能把某一制式说成全球唯一；按题干指定标准作答。

### 回忆卡 14｜电磁波谱与光纤

先答：电磁波频率怎样排序？光纤常用窗口和单模光纤特点是什么？

- 🔴 频率由低到高：无线电/微波 → 红外 → 可见光 → 紫外 → X射线 → γ射线，波长反向；常用光纤窗口为`1.31 μm、1.55 μm`。
- 🔵 光纤题先由波长/频率关系排序，再从损耗、色散、带宽、距离解释窗口或光纤类型选择。
- ⚫ 1.55 μm附近损耗更低，1.31 μm附近有传统色散优势；单模带宽高、色散小，适合长距离高速传输。

## 七、高频电子线路

### 回忆卡 15｜高频小信号调谐放大器

先答：调谐回路完成哪些作用？Q值怎样影响带宽和选择性？

- 🔴 调谐回路用于选频、阻抗变换和负载；单调谐近似`BW≈f0/Q`，Q越高通常选择性越好、带宽越窄。
- 🔵 由谐振频率和Q求带宽，再结合级联级数判断总增益与总选择性。
- ⚫ 多级级联虽提高选择性，但总带宽可能变窄；实际Q受有载条件影响。

### 回忆卡 16｜高频功率放大器

先答：丙类谐振功放为何效率高？欠压、临界、过压怎样判？

- 🔴 丙类导通角小于180°，集电极电流为脉冲，谐振回路选出基波近似正弦；最佳工作常在临界附近。
- 🔵 结合负载线、集电极电压和电流脉冲形状，按欠压 → 临界 → 过压判工作状态及负载变化。
- ⚫ 具体动态线和临界条件依教材模型；不能把“小导通角”误认为输出也是脉冲。

### 回忆卡 17｜混频与超外差接收

先答：混频怎样得到中频？超外差链路和镜像干扰是什么？

- 🔴 非线性混频产生和差频，选固定中频`fIF=|fLO−fRF|`；链路为天线/高放 → 混频+本振 → 中放 → 检波/解调 → 低放。
- 🔵 给定本振注入方式先求`fLO`，再解所有满足同一中频的射频，识别信号频率与镜像频率。
- ⚫ 另一射频也能与本振产生同一中频即镜像；提高混频前高频选择性可抑制它。

### 回忆卡 18｜振荡器与锁相环

先答：反馈振荡条件是什么？PLL由哪些模块组成并能完成什么功能？

- 🔴 振荡稳态满足`|AF|=1`且总相移`2kπ`，起振时环路增益略大于1；PLL由鉴相器、环路滤波器、压控振荡器组成。
- 🔵 振荡题先判相位再算幅值；PLL题沿“相位误差 → 滤波控制电压 → VCO频率变化 → 锁定”解释。
- ⚫ 锁定后输出频率/相位跟踪输入，可用于频率合成、解调和载波恢复；复杂捕获范围与稳定性按题目模型计算。
