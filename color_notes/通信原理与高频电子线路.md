# 通信原理与高频电子线路｜三色回忆卡

先口述再核对：红色写结论，蓝色写原因或过程链，黑色写适用条件、代价和易混边界。

## 一、通信系统与传输方式

### 回忆卡1：通信系统组成

先答：一个典型通信系统由哪些基本部分组成？


- 🔴 信源、发送设备、信道、接收设备、信宿，并受噪声影响。
- 🔵 按信息流向依次回答，再用一句说明各环节作用。
- ⚫ 消息、信息和信号相关但不是同义词；信号是信息的物理载体。
### 回忆卡2：模拟通信与数字通信

先答：模拟通信和数字通信有什么主要区别？


- ⚫ 模拟通信处理连续参数；数字通信传送离散符号。
- 🔵 数字通信优点写抗干扰较强、便于再生、加密、存储和复用。
- 🔴 数字通信仍会受噪声影响并出现误码，也可能使用模拟载波。
### 回忆卡3：单工、半双工、全双工

先答：三种通信方式如何区分，各举一个例子。


- 🔴 单工只单向；半双工双向但不同时；全双工可同时双向。
- 🔵 例子依次为广播、对讲机、电话。
- ⚫ 对讲机双方都能发言，但不能同时发射，所以不是全双工。
## 二、指标与基本公式

### 回忆卡4：码元速率与信息速率

先答：Baud和bit/s分别表示什么？


- ⚫ Baud表示每秒码元数；bit/s表示每秒比特数。
- 🔵 二进制时一个码元通常带1 bit，多进制时一个码元可带多个bit。
- 🔴 两个速率的数值不总是相等，不能把单位混用。
### 回忆卡5：香农公式

先答：香农公式是什么，各变量说明什么？


- ⚫ `C=B log₂(1+S/N)`；C为容量、B为带宽、S/N为线性信噪比。
- 🔵 结论是增大带宽或信噪比可提高理论容量。
- 🔴 容量是理论上限；信噪比收益呈对数增长，不是线性增长。
### 回忆卡6：抽样定理

先答：为避免混叠，抽样频率与信号最高频率应满足什么关系？


- 🔴 理想条件下 `fs≥2fH`。
- 🔵 先找最高频率，再乘2；实际系统通常保留过渡带余量。
- ⚫ 抽样只离散时间，量化才把连续幅值变成有限等级。
## 三、调制与数字化

### 回忆卡7：为什么要调制

先答：通信系统为什么需要调制？


- 🔴 适应信道和天线、实现远距离传输、便于频分复用并改善性能。
- 🔵 简答列三点即可，再说明调制是改变载波参数。
- ⚫ 混频主要搬移中心频率，解调才是恢复原消息。
### 回忆卡8：AM、FM、PM

先答：AM、FM、PM分别改变载波的什么参数？


- 🔵 AM改幅度，FM改频率，PM改相位；一般条件下FM抗幅度干扰强于AM。
- 🔴 AM的信息直接在幅度上，幅度噪声会污染包络；FM可先限幅削弱幅度波动，再从频率变化恢复信息。
- ⚫ FM并非不受噪声影响，弱信号时也会恶化，而且通常占用更宽带宽、设备更复杂。
### 回忆卡9：ASK、FSK、PSK、QAM

先答：四种数字调制分别利用载波什么变化表示信息？


- ⚫ ASK幅度、FSK频率、PSK相位、QAM幅度和相位共同变化。
- 🔵 先看字母A/F/P，再判断载波参数。
- 🔴 数字调制传数字信息，但信道中的载波波形仍是连续时间信号。
### 回忆卡10：抽样、量化、编码

先答：模拟信号数字化的三个基本步骤是什么？


- ⚫ 抽样、量化、编码。
- 🔵 抽样离散时间，量化离散幅值，编码表示成二进制代码。
- 🔴 量化会产生量化误差，位数增加通常能提高分辨率。
### 回忆卡11：FDM、TDM、WDM与CDM

先答：四种复用方式分别用什么资源区分多路信号？


- ⚫ FDM分频率，TDM分时隙，WDM分光波长，CDM分码字。
- 🔵 FDM、WDM可让多路同时传输；TDM轮流占用时隙；CDM依靠匹配扩频码分离用户。
- 🔴 WDM主要用于光纤；TDM需要时隙同步；CDM不是给每路固定分配不同频带。
### 回忆卡12：NRZ与曼彻斯特码

先答：NRZ和曼彻斯特码最小区别是什么？


- ⚫ NRZ每位不要求回零或跳变；曼彻斯特码每位中间必有跳变，可同时提供时钟信息。
- 🔵 NRZ带宽利用较好但连续相同比特时不利于同步；曼彻斯特自同步较好但通常占用更宽带宽。
- 🔴 不同约定的跳变方向可能代表不同数据，辨认曼彻斯特应抓“位中跳变”，不要只背方向。
## 四、介质、波谱与移动通信

### 回忆卡13：双绞线、同轴和光纤

先答：三种有线介质各有什么突出特点？


- ⚫ 双绞线便宜易布线；同轴屏蔽较好；光纤容量大、损耗低、抗电磁干扰。
- 🔵 简答从成本、带宽、抗干扰、距离四方面比较。
- 🔴 光纤也有接续和光电转换成本，不能说在所有场景都最便宜。
### 回忆卡14：电磁波谱排序

先答：从无线电波到紫外线，频率由低到高怎样排列？


- 🔵 无线电波→微波→红外线→可见光→紫外线。
- 🔴 频率升高时波长变短，可用这一关系反向检查。
- ⚫ 微波属于无线电波的一部分，题目并列时按更具体频段比较。
### 回忆卡15：光纤通信窗口

先答：1310 nm和1550 nm窗口各有什么突出特点？


- 🔴 1310 nm色散较小；1550 nm损耗最低、适合长距离高速传输。
- 🔵 口诀：“1310看色散，1550看损耗”。
- ⚫ 两者都是近红外窗口，不是人眼可见光。
### 回忆卡16：1G到5G

先答：1G到5G各有什么代表性特点？


- ⚫ 1G模拟语音，2G数字语音短信，3G增强数据，4G全IP宽带，5G宽带、海量连接和低时延。
- 🔵 按“模拟→数字→数据→宽带IP→三大场景”记忆。
- 🔴 Wi‑Fi不是某一代蜂窝移动通信，代际也不只由下载速度决定。
## 五、高频接收电路

### 回忆卡17：超外差接收机

先答：超外差接收机有哪些主要环节？


- 🔵 高频选频放大、混频、中频放大、解调、低频放大。
- 🔴 按信号流向回答，并指出混频把不同射频变到固定中频。
- ⚫ 混频不直接恢复消息；解调才负责恢复基带信号。
### 回忆卡18：锁相环

先答：锁相环由哪三个基本部分组成，有什么用途？


- ⚫ 鉴相器、环路滤波器、压控振荡器；用于同步、频率合成和解调等。
- 🔵 简答按“比较相位—滤除控制误差—调整振荡频率”说明闭环过程。
- 🔴 振荡器仍需要直流电源提供能量，并非无能量输入产生信号。
### 回忆卡19：现实信号怎样进入计算机

先答：声音怎样变成计算机可处理的数据？


- ⚫ 声音→麦克风→模拟电信号→放大/滤波→ADC抽样、量化、编码→数字数据。
- 🔵 抽样离散时间，量化离散幅值，编码用二进制表示量化等级。
- 🔴 传感器先得到模拟电信号；数字信息是表示方式，电信号是物理载体，两者不是同一概念。
### 回忆卡20：计算机为什么常用高低电平

先答：为什么计算机内部常用高、低电平表示0和1？


- 🔵 半导体开关适合高速、低成本、大规模集成，离散电平便于判决和恢复。
- 🔴 有噪声时只要扰动未越过判决阈值，逻辑状态仍可正确识别，因此比精确保留连续幅值更稳健。
- ⚫ 逻辑0和1不固定等于0 V和1 V；内部互连也可能使用光或其他物理载体，不能把结论绝对化。
### 回忆卡21：同步解决什么问题

先答：数字通信为什么需要位同步和帧同步？


- 🔴 位同步确定何时判决每个码元，帧同步确定一组数据的起止边界。
- 🔵 没有正确时序，即使电平都收到，也可能因采样时刻或分组错位而解释错误。
- ⚫ 通信同步不等于业务数据一致性；订单系统仍需序号、事务、幂等和数据库机制。
## 六、编码、眼图与光链路

### 回忆卡22：信源编码与信道编码

先答：二者目的有什么不同？


- 🔵 信源编码去冗余以压缩；信道编码加受控冗余以检错或纠错。
- 🔴 奇偶校验和CRC主要检错，纠错码在能力范围内可直接纠错，ARQ依靠反馈重传。
- ⚫ CRC不能保证零差错，也通常不能单独纠错；压缩率高不代表抗干扰强。
### 回忆卡23：怎样读眼图

先答：眼图张开和闭合分别说明什么？


- ⚫ 眼睛越开通常判决余量越大；闭合提示噪声、失真、码间串扰或同步误差加重。
- 🔵 位同步决定判决时刻，帧同步决定数据块边界，载波同步支持相干解调。
- 🔴 眼图只能综合反映链路质量，不能仅凭一张图唯一确定具体坏件。
### 回忆卡24：光纤链路的电光转换

先答：光纤发送端和接收端分别完成什么转换？


- 🔵 发送端电转光，光纤传输，接收端光电二极管完成光转电，再放大判决。
- 🔴 衰减降低光功率，色散展宽脉冲并可能加重码间串扰。
- ⚫ 1550 nm仍不可见；检修时不能直视可能有光的端面。
### 回忆卡25：光链路三种仪表

先答：光功率计、稳定光源和OTDR分别适合做什么？

- 🔵 光功率计测光功率；光源配功率计测插损；OTDR估计事件位置和故障距离。
- 🔴 无光先查光源和连接，功率低再查脏污、弯曲、熔接和链路损耗。
- ⚫ OTDR有近端盲区，结果需结合链路长度和连接资料，不能把任意反射峰都当断纤。
