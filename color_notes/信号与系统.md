# 信号与系统：三色笔记

三色按考频和出题稳定性划分：红色优先背熟，蓝色用于补齐得分点，黑色用于防漏和边界判断。

## 🔴 高频考点

- 🔴 **系统五大性质**：线性、时不变、因果、稳定、可逆。常见考法：给定表达式判断性质。易错边界：线性与时不变彼此独立；BIBO稳定指有界输入得到有界输出。
- 🔴 **抽样定理**：`fs≥2fm`。常见考法：求最低抽样频率、判断混叠。易错边界：抽样率解决时间离散问题，ADC位数解决幅值分辨问题，二者不能互替。
- 🔴 **傅里叶变换**：时域转换到频域，用于频谱分析。常见考法：判断变换方向和频域意义。易错边界：频谱是同一信号的另一种表示，不是额外独立信号。
- 🔴 **正弦信号三要素**：`x(t)=Am sin(ωt+φ)`，并有 `ω=2πf`、`T=1/f`。常见考法：由表达式读幅值、频率、初相位。易错边界：角频率单位是rad/s，不是Hz。
- 🔴 **奈奎斯特频率**：可无混叠表示的最高频率为采样频率的一半，即 `fs/2`。常见考法：由采样率判断可分析的最高频率。易错边界：它是采样系统的频率上限，不等同于任意实际信号都能无失真恢复。
- 🔴 **LTI系统响应**：线性时不变系统的零状态响应由输入与冲激响应卷积得到。常见考法：写出 `y=x*h` 并说明各量。易错边界：含初始储能时，总响应还包括零输入响应。

## 🔵 中频考点

- 🔵 **卷积**：LTI系统的零状态输出为输入与冲激响应的卷积。常见考法：解释卷积物理意义、判断卷积性质。易错边界：含初始储能时，总响应不只包含零状态响应。
- 🔵 **拉普拉斯与Z变换**：拉普拉斯用于连续系统的复频域分析，Z变换用于离散系统。常见考法：对应变换和用途。易错边界：两者都要结合收敛域、初始条件理解。
- 🔵 **因果性**：当前输出不得依赖未来输入。常见考法：判断移位表达式。易错边界：`y(t)=x(t+1)` 需要未来输入，属于非因果；`y(t)=x(t-1)` 属于因果。
- 🔵 **冲激响应**：LTI系统对单位冲激输入的输出记为 `h(t)` 或 `h[n]`，它与输入卷积可得到零状态响应。常见考法：说明冲激响应的作用。易错边界：冲激响应刻画的是系统特性，不是任意输入信号本身。
- 🔵 **频率响应**：LTI系统频率响应描述各频率分量通过系统后的幅度和相位变化。常见考法：判断滤波器保留或衰减哪些频率。易错边界：幅频响应和相频响应共同决定输出，不能只看增益大小。

## ⚫ 低频了解

- ⚫ **偶、奇信号**：偶信号满足 `x(t)=x(-t)`，奇信号满足 `x(t)=-x(-t)`。常见考法：按表达式判断对称性。易错边界：零信号既可视作偶信号，也可视作奇信号。
- ⚫ **时域与频域关系**：时域卷积对应频域相乘。常见考法：说明滤波为什么可以在频域理解。易错边界：这一对应关系有适用的变换存在条件。
- ⚫ **周期与频率**：周期 `T` 与频率 `f` 互为倒数。常见考法：在秒、毫秒、赫兹和千赫兹之间换算。易错边界：只有周期信号才有基本周期和基本频率。

## 考前公式与简答

### 必会公式

- **正弦信号**：`x(t)=Am sin(ωt+φ)`。`Am` 是幅值，`ω=2πf` 是角频率(rad/s)，`f` 是频率(Hz)，`φ` 是初相位，周期 `T=1/f`。
- **卷积响应**：连续时间 `y(t)=∫x(τ)h(t-τ)dτ`，离散时间 `y[n]=Σx[k]h[n-k]`。`x` 是输入，`h` 是冲激响应，`y` 是LTI系统零状态输出。
- **抽样条件**：`fs≥2fm`。`fs` 是采样频率，`fm` 是信号最高频率；不满足时不同频率成分会重叠，产生混叠。
- **变换对应**：傅里叶变换把时域变到频域；拉普拉斯变换用于连续系统的 `s` 域分析；Z变换用于离散系统的 `z` 域分析。变换式还要结合收敛条件。

### 常见简答题

- **怎样判断系统因果与稳定？** 因果系统当前输出只依赖当前及过去输入；BIBO稳定要求任何有界输入都产生有界输出。二者是独立性质。
- **傅里叶变换的意义是什么？** 它把复杂信号分解为不同频率的正弦成分，便于观察频谱、带宽和滤波效果；时域与频域是同一信号的两种表示。
